1. Install Production Dependencies:
   cd .next/standalone
   npm install --production --legacy-peer-deps

2. Configure Environment (.next/standalone/.env):
   DATABASE_URL=file:/absolute/path/to/.next/standalone/prisma/dev.db
   NEXTAUTH_URL=http://localhost:3000
   NEXTAUTH_SECRET=generate-a-random-secret-key-here

3. Copy Database:
   cp -r prisma .next/standalone/
   
   Or for production with PostgreSQL:
   DATABASE_URL=postgresql://user:password@localhost:5432/thaiaccounting

4. Start Server:
   cd .next/standalone
   npm start

   Server will run on: http://localhost:3000
