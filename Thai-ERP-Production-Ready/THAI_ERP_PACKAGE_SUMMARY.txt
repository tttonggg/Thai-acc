🚀 QUICK START
═══════════════════════════════════════════════════════════════════════════════

Method 1: Automatic (Recommended)
─────────────────────────────────
1. unzip Thai-ERP-Production-Ready.zip
2. cd Thai-ERP-Production-Ready
3. ./QUICKSTART.sh
4. npm run dev
5. Open http://localhost:3000

Method 2: Manual
────────────────
1. unzip Thai-ERP-Production-Ready.zip
2. cd Thai-ERP-Production-Ready
3. cp config/* ./
4. npm install
5. npm run db:generate
6. npm run db:push
7. npx prisma db seed
8. npm run dev
9. Open http://localhost:3000
