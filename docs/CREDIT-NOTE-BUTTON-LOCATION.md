## 🎯 ตำแหน่งปุ่ม "สร้างใบลดหนี้"

**เมื่อมีข้อมูลแล้ว** ปุ่มจะอยู่ที่:

```
┌─────────────────────────────────────────────────────────────┐
│                                                              │
│  ใบลดหนี้ (Credit Notes)            [สร้างใบลดหนี้]  ←─ ปุ่มอยู่ตรงนี้!
│  จัดการใบลดหนี้สำหรับลูกค้า           (+ ไอคอนสีแดง)     │
│                                                              │
└─────────────────────────────────────────────────────────────┘
         ↑ มุมขวาบน (Top Right) ของหน้าจอ
```

## 🔍 วิธีค้นหาปุ่ม

### 1. เข้าสู่หน้าใบลดหนี้

- คลิกเมนู **"ใบลดหนี้ (CN)"** ใน Sidebar ด้านซ้าย
- หรือเข้า URL: `http://localhost:3000/credit-notes`

### 2. ดูที่มุมขวาบน

- ปุ่มจะอยู่ **ด้านบนขวา** ของหน้าจอ
- เป็นปุ่ม **สีแดง** มีไอคอน **(+)**
- ข้อความว่า **"สร้างใบลดหนี้"**

### 3. ถ้าไม่เห็นปุ่ม

เป็นเพราะ **ยังไม่มีข้อมูลใบลดหนี้** (ตอนนี้มี 0 รายการ)

- ข้อความ "ไม่พบข้อมูลใบลดหนี้" จะแสดง
- แต่ปุ่ม "สร้างใบลดหนี้" ควรจะอยู่ด้านบน

## 🐛 ถ้ายังไม่เห็นปุ่ม

### ตรวจสอบ:

1. **ล็อกอินเข้าสู่ระบบหรือยัง?**
   - ไปที่: http://localhost:3000
   - ล็อกอินด้วย admin@thaiaccounting.com

2. **URL ถูกต้องหรือยัง?**
   - ควรเป็น: `http://localhost:3000/credit-notes`

3. **ทำหน้าจอ Refresh (F5)**
   - กด F5 หรือ Cmd+R (Mac)

4. **เช็ค Console Errors**
   - กด F12 → แท็บ Console
   - ดูว่ามี error อะไรหรือไม่

## 💡 สาเหตุที่อาจไม่เห็นปุ่ม

1. **Browser Cache** - ลอง Hard Refresh (Cmd+Shift+R)
2. **Loading State** - รอสักครู่ให้โหลดข้อมูล
3. **Error Loading** - มี error ใน Console
4. **Not Logged In** - ต้องล็อกอินก่อน
