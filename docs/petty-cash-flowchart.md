## Error Handling

```
Common Errors:

1. Insufficient Fund Balance
   Error: "а№ҖаёҮаёҙаёҷаёӘаё”аёўа№Ҳаёӯаёўа№„аёЎа№Ҳа№ҖаёһаёөаёўаёҮаёһаёӯ (аё„аёҮа№Җаё«аёҘаё·аёӯ аёҝ1,000)"
   Solution: Reimburse fund first

2. Already Approved
   Error: "а№ғаёҡа№ҖаёҡаёҙаёҒа№ҖаёҮаёҙаёҷаёӘаё”аёўа№Ҳаёӯаёўаёҷаёөа№үа№„аё”а№үаёЈаёұаёҡаёҒаёІаёЈаёӯаёҷаёёаёЎаёұаё•аёҙа№ҒаёҘа№үаё§"
   Solution: Check journal entry instead

3. Already Reimbursed
   Error: "а№ғаёҡа№ҖаёҡаёҙаёҒа№ҖаёҮаёҙаёҷаёӘаё”аёўа№Ҳаёӯаёўаёҷаёөа№үа№„аё”а№үаёЈаёұаёҡаёҒаёІаёЈа№Җаё•аёҙаёЎа№ҖаёҮаёҙаёҷа№ҒаёҘа№үаё§"
   Solution: Cannot reimburse twice
```

## Accounting Equations

### Double-Entry Verification

**Approval:**

```
Assets (Petty Cash Fund) вҶ“ by voucher amount
Expenses вҶ‘ by voucher amount

Accounting Equation:
Assets = Liabilities + Equity + Revenue - Expenses
-1,000 = 0 + 0 + 0 - 1,000  вң“ Balanced
```

**Reimbursement:**

```
Assets (Petty Cash Fund) вҶ‘ by voucher amount
Assets (Cash/Bank) вҶ“ by voucher amount

Accounting Equation:
Assets = Liabilities + Equity + Revenue - Expenses
+1,000 - 1,000 = 0  вң“ Balanced
```
