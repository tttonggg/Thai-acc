# Video Tutorial Script: 09 - Backup and Restore

## Video Information

- **Title:** Backup and Restore (การสำรองและกู้คืนข้อมูล)
- **Duration:** 10-12 minutes
- **Target Audience:** Administrators
- **Prerequisites:** Admin access

## Introduction (1 minute)

**[Scene: Backup management page]**

**Narrator:** "สวัสดีครับ/ค่ะ ในตอนนี้เราจะมาเรียนรู้การสำรองและกู้คืนข้อมูล
ซึ่งเป็นสิ่งสำคัญมากในการป้องกันข้อมูลสูญหาย"

**On-screen text:**

- การสำรองข้อมูล (Backup)
- การกู้คืนข้อมูล (Restore)
- การสำรองอัตโนมัติ (Auto Backup)

## Section 1: Why Backup is Important (1 minute)

**[Scene: Warning signs]**

**Narrator:** "เหตุผลที่ต้องสำรองข้อมูล"

**Risks:**

- 💥 ฮาร์ดดิสก์เสีย
- 🦠 ไวรัสโจมตี
- 👤 ลบข้อมูลโดยไม่ตั้งใจ
- 🔌 ไฟฟ้าดับขณะทำงาน
- 🔄 อัปเดตระบบผิดพลาด

**3-2-1 Backup Rule:**

```
3: สำรอง 3 ชุด
2: เก็บใน 2 สื่อที่ต่างกัน
1: เก็บ 1 ชุดนอกสถานที่
```
