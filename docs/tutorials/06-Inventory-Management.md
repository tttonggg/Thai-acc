# Video Tutorial Script: 06 - Inventory Management

## Video Information
- **Title:** Inventory Management (การจัดการสินค้าคงคลัง)
- **Duration:** 15-18 minutes
- **Target Audience:** Inventory managers, accountants
- **Prerequisites:** Basic understanding of products

## Introduction (1 minute)

**[Scene: Inventory dashboard]**

**Narrator:**
"สวัสดีครับ/ค่ะ ในตอนนี้เราจะมาเรียนรู้การจัดการสินค้าคงคลัง ตั้งแต่การรับสินค้า จ่ายสินค้า ไปจนถึงการคำนวณต้นทุน"

**On-screen text:**
- FIFO (First In, First Out)
- Weighted Average (ค่าเฉลี่ยถ่วงน้ำหนัก)
- Stock Valuation

## Section 1: Inventory Overview (2 minutes)

**[Scene: Stock balance report]**

**Narrator:**
"ระบบสินค้าคงคลังช่วยติดตามจำนวนและมูลค่าสินค้าแบบ Real-time"

**Key Features:**
1. ติดตามจำนวนสินค้า
2. คำนวณต้นทุนสินค้า
3. แจ้งเตือนสินค้าใกล้หมด
4. รายงานการเคลื่อนไหว

**Costing Methods:**
- **FIFO** - ขายของที่เข้าก่อนก่อน
- **Weighted Average** - ใช้ต้นทุนเฉลี่ย

```
ต้นทุนต้นงวด: 25,000
```

## Section 3: Goods Receipt (4 minutes)

**[Scene: Create goods receipt]**

**Narrator:**
"บันทึกการรับสินค้าจากผู้ขาย"

**Goods Receipt:**
```
เลขที่: GR-202603-001
วันที่: 05/03/2026
ผู้ขาย: บริษัท ซัพพลายเออร์ จำกัด
เลขที่อ้างอิง: PO-12345
```

**Receipt Lines:**
```
สินค้า          จำนวน  ราคาต่อหน่วย  รวม
--------------------------------------------------
PROD-001        30     25,000       750,000
PROD-002        50     15,000       750,000
--------------------------------------------------
รวม                              1,500,000
```

**[Scene: Add lines and save]**

**Narrator:**
"เมื่อโพสต์ สินค้าจะเข้าสต็อกและต้นทุนจะถูกบันทึก"

**Journal Entry:**
```
เดบิต  สินค้าคงเหลือ        1,500,000
เครดิต เจ้าหนี้การค้า        1,500,000
```

## Section 4: Goods Issue (3 minutes)

**[Scene: Create goods issue]**

**Narrator:**
"บันทึกการจ่ายสินค้าออกจากคลัง"

**Goods Issue:**
```
เลขที่: GI-202603-001
วันที่: 10/03/2026
ประเภท: ขายสินค้า
อ้างอิง: INV-202603-001
```

**Issue Lines:**
```
สินค้า          จำนวน  ต้นทุน/หน่วย  รวมต้นทุน
--------------------------------------------------
PROD-001        20     25,000       500,000
```

**FIFO Calculation:**
```
เข้า  01/03: 50 หน่วย @ 25,000 = 1,250,000
เข้า  05/03: 30 หน่วย @ 25,000 = 750,000
-----------------------------------------
รวม: 80 หน่วย, มูลค่า 2,000,000

ออก  10/03: 20 หน่วย @ 25,000 = 500,000 (จากล็อต 01/03)
เหลือ: 60 หน่วย @ 25,000 = 1,500,000
```

**[Scene: Show cost calculation]**

## Section 5: Stock Adjustments (2 minutes)

**[Scene: Stock adjustment]**

**Narrator:**
"ปรับปรุงสต็อกเมื่อพบความผิดพลาด"

**Adjustment Scenarios:**

- สินค้าเสียหาย

```

จำนวนที่นับได้: 28

เหตุผล: สินค้าเสียหาย
```

```
เครดิต สินค้าคงเหลือ             50,000
```
