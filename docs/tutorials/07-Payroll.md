# Video Tutorial Script: 07 - Payroll

## Video Information

- **Title:** Payroll Management (การจัดการเงินเดือน)
- **Duration:** 15-18 minutes
- **Target Audience:** HR staff, accountants
- **Prerequisites:** Company settings configured

## Introduction (1 minute)

**[Scene: Payroll dashboard]**

**Narrator:** "สวัสดีครับ/ค่ะ ในตอนนี้เราจะมาเรียนรู้การจัดการเงินเดือน
ซึ่งรวมถึงการคำนวณประกันสังคม และภาษีหัก ณ ที่จ่าย"

**On-screen text:**

- ประกันสังคม (Social Security)
- ภาษีเงินได้ (PND1)
- เงินเดือนสุทธิ (Net Salary)

## Section 2: Payroll Components (3 minutes)

**Narrator:** "เข้าใจองค์ประกอบของเงินเดือน"


## Section 3: Processing Payroll (4 minutes)

**[Scene: Payroll processing]**

**Narrator:** "ประมวลผลเงินเดือนประจำเดือน"

**Step 1: Select Period**

```
เดือน: มีนาคม 2026
```

**Step 2: Calculate Payroll**

**[Scene: Click calculate]**

**Calculation Results:**

```
พนักงาน: สมชาย ใจดี

รายได้:
  เงินเดือน              45,000
  ค่าล่วงเวลา              5,000
  ค่าเดินทาง               1,500
  ค่าอาหาร                2,000
  -------------------------
  รวม                    53,500

หัก:
  ประกันสังคม (5%)          750  (เพดาน 15,000)
  ภาษีเงินได้             1,850
  -------------------------
  รวมหัก                  2,600

เงินเดือนสุทธิ:          50,900
```

**Tax Calculation:**

```
เงินได้สุทธิ:           53,500 - 750 = 52,750
หักค่าใช้จ่าย 50%:      26,375 (ไม่เกิน 100,000)
หักค่าลดหย่อน:
  ตนเอง:                60,000
  คู่สมรส:               60,000
  บุตร:                  30,000 x 2 = 60,000
  ประกันสังคม:             750
  -------------------------
  รวมลดหย่อน:          206,750

เงินได้พึงประเมิน:      52,750 - 26,375 - 206,750 = -180,375
ภาษี: 0 บาท
```

**[Scene: Show tax calculation details]**
