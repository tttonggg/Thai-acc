## Section 1: Login Issues (2 minutes)

**[Scene: Login error]**

**Problem 1: Cannot Login**

```
ข้อผิดพลาด: "อีเมลหรือรหัสผ่านไม่ถูกต้อง"
```

**Solutions:**

1. ตรวจสอบ Caps Lock
2. ตรวจสอบภาษาพิมพ์ (Thai/English)
3. ลองรีเซ็ตรหัสผ่าน
4. ติดต่อผู้ดูแลระบบ

**[Scene: Password reset]**

**Password Reset Steps:**

```
1. คลิก "ลืมรหัสผ่าน"
2. ใส่อีเมล
3. ตรวจสอบอีเมล
4. คลิกลิงก์รีเซ็ต
5. ตั้งรหัสผ่านใหม่
```

**Problem 2: Session Expired**

```
ข้อผิดพลาด: "เซสชันหมดอายุ กรุณาเข้าสู่ระบบใหม่"
```

**Solution:**

- ออกจากระบบและเข้าใหม่
- ล้างคุกกี้เบราว์เซอร์
- ใช้โหมดไพรเวต

## Section 2: Document Issues (3 minutes)

**[Scene: Invoice error]**

**Problem 3: Cannot Post Invoice**

```
ข้อผิดพลาด: "ไม่สามารถออกใบกำกับภาษีได้"
```

**Common Causes:**

1. ❌ ไม่ได้เลือกลูกค้า
2. ❌ ไม่มีรายการสินค้า
3. ❌ จำนวนสินค้าไม่พอ
4. ❌ วันที่ไม่ถูกต้อง

**[Scene: Check each item]**

**Problem 4: Unbalanced Journal Entry**

```
ข้อผิดพลาด: "ยอดเดบิตไม่เท่ากับยอดเครดิต"

เดบิตรวม:  50,000
เครดิตรวม: 45,000
ผลต่าง:    5,000 ❌
```

**Solution:**

```
ตรวจสอบแต่ละรายการ:
รายการ 1: ลูกหนี้การค้า เดบิต 50,000 ✓
รายการ 2: รายได้      เครดิต 45,000 ✓
         ภาษีขาย     เครดิต  3,500 ← ขาด!
```

**[Scene: Fix the entry]**

## Section 3: Data Issues (2 minutes)

**[Scene: Data error]**

**Problem 5: Duplicate Code**

```
ข้อผิดพลาด: "รหัสนี้มีอยู่แล้ว"

รหัสลูกค้า: CUST-001
สถานะ: ซ้ำ
```

**Solutions:**

1. ใช้รหัสใหม่
2. หรืออัปเดตข้อมูลเดิม
3. ตรวจสอบในรายการที่ถูกลบ

**[Scene: Find duplicate]**

**Problem 6: Data Not Saving**

```
ข้อผิดพลาด: "ไม่สามารถบันทึกข้อมูลได้"
```

**Checklist:**

- ☑ อินเทอร์เน็ตเชื่อมต่อ
- ☑ ฟิลด์ที่จำเป็นครบ
- ☑ รูปแบบข้อมูลถูกต้อง
- ☑ ไม่มีช่องว่างพิเศษ

## Section 4: Report Issues (2 minutes)

**[Scene: Report error]**

**Problem 7: Report Shows Wrong Data**

**Solutions:**

1. ตรวจสอบช่วงวันที่
2. ตรวจสอบสถานะเอกสาร (ต้องโพสต์แล้ว)
3. รีเฟรชรายงาน
4. ล้างแคช

**[Scene: Check document status]**

**Problem 8: Export Fails**

```
ข้อผิดพลาด: "ไม่สามารถส่งออกไฟล์ได้"
```

**Solutions:**

- ลดจำนวนข้อมูล
- ลองรูปแบดอื่น (CSV แทน Excel)
- ตรวจสอบพื้นที่ดิสก์

## Section 5: Performance Issues (2 minutes)

**[Scene: Slow loading]**

**Problem 9: System Running Slow**

**Solutions:**

```
1. ล้างแคชเบราว์เซอร์
   Chrome: Ctrl+Shift+Delete

2. ปิดแท็บที่ไม่ใช้

3. ลดช่วงวันที่ในรายงาน
   (แทน 1 ปี ใช้ 1 เดือน)

4. ตรวจสอบการเชื่อมต่ออินเทอร์เน็ต
   (ควร 10 Mbps ขึ้นไป)
```

**[Scene: Clear cache]**

## Section 6: Print Issues (2 minutes)

**[Scene: Print error]**

**Problem 10: Cannot Print**

**Solutions:**

1. ตรวจสอบเครื่องพิมพ์
2. ตรวจสอบไดรเวอร์
3. ลองพิมพ์เป็น PDF ก่อน
4. ใช้ Print Preview
