# Video Tutorial Script: 01 - Getting Started

## Video Information
- **Title:** Getting Started with Thai Accounting ERP
- **Duration:** 10-12 minutes
- **Target Audience:** New users, business owners, accountants
- **Language:** Thai with English subtitles available

## Introduction (1 minute)

**[Scene: Dashboard view with company logo]**

**Narrator:**
"สวัสดีครับ/ค่ะ ยินดีต้อนรับสู่ Thai Accounting ERP โปรแกรมบัญชีมาตรฐานไทยที่ครบครันด้วยฟีเจอร์ที่ธุรกิจ SME ไทยต้องการ ในวิดีโอนี้ เราจะพาคุณรู้จักกับระบบและเริ่มต้นใช้งานอย่างถูกต้อง"

**On-screen text:**
- Thai Accounting ERP
- โปรแกรมบัญชีสำหรับธุรกิจไทย

## Section 1: System Overview (2 minutes)

**[Scene: Login page → Dashboard]**

**Narrator:**
"Thai Accounting ERP คือโปรแกรมบัญชีบนคลาวด์ที่ออกแบบมาสำหรับธุรกิจไทยโดยเฉพาะ ระบบรองรับมาตรฐานบัญชีไทย (TFRS) และกฎหมายภาษีไทยทั้งหมด"

**Key Features Highlight:**
1. ผังบัญชีมาตรฐานไทย
2. ระบบภาษีมูลค่าเพิ่ม 7%
3. ระบบภาษีหัก ณ ที่จ่าย ภ.ง.ด.3 และ ภ.ง.ด.53
4. การจัดการสต็อกสินค้า
5. ระบบเงินเดือนและประกันสังคม
6. รายงานทางการเงินครบถ้วน

**[Scene: Navigate through main menu]**

**Narrator:**
"เมนูหลักของระบบจะอยู่ทางด้านซ้ายมือ แบ่งออกเป็นหมวดหมู่ต่างๆ ได้แก่ บัญชี ธนาคาร สินค้าคงคลัง สินทรัพย์ถาวร เงินเดือน และรายงาน"

## Section 3: Initial Setup (3 minutes)

**[Scene: Settings page]**

**Narrator:**
"ก่อนเริ่มใช้งานจริง เราต้องตั้งค่าข้อมูลบริษัทก่อน เข้าไปที่เมนู ตั้งค่า → ข้อมูลบริษัท"

**Step-by-step setup:**

**Step 1: Company Information**
```
ชื่อบริษัท: บริษัท ตัวอย่าง จำกัด
ชื่อบริษัท (EN): Example Company Ltd.
เลขประจำตัวผู้เสียภาษี: 1234567890123
ที่อยู่: 123 ถนนสุขุมวิท แขวงคลองเตย
โทรศัพท์: 02-123-4567
อีเมล: info@example.com
```

**[Scene: Fill in the form]**

**Narrator:**
"กรอกข้อมูลให้ครบถ้วน เพราะข้อมูลนี้จะปรากฏในเอกสารต่างๆ ของบริษัท เช่น ใบกำกับภาษี ใบเสร็จรับเงิน"

**Step 2: Accounting Settings**
```
อัตราภาษีมูลค่าเพิ่ม: 7%
วันเริ่มปีงบประมาณ: 01/01
สกุลเงิน: บาท (THB)
รูปแบบวันที่: วัน/เดือน/ปี
```

**Step 3: Save Settings**

**[Scene: Click save button, show success message]**

**Narrator:**
"คลิกที่ปุ่ม บันทึก เมื่อกรอกข้อมูลครบแล้ว ระบบจะบันทึกการตั้งค่าและพร้อมใช้งาน"

## Section 4: User Roles (2 minutes)

**[Scene: User management page]**

**Narrator:**
"ระบบมีการกำหนดสิทธิ์ผู้ใช้งาน 4 ระดับ"

**Role Explanation:**

| Role | สิทธิ์ | เหมาะสำหรับ |
|------|--------|-------------|
| ADMIN | ทุกอย่าง | เจ้าของกิจการ, IT |
| ACCOUNTANT | บัญชีทั้งหมด | นักบัญชี |
| USER | สร้างเอกสาร | พนักงานทั่วไป |
| VIEWER | ดูอย่างเดียว | ผู้บริหาร |

**[Scene: Show user creation]**

**Narrator:**
"ผู้ดูแลระบบสามารถเพิ่มผู้ใช้งานใหม่ได้โดยไปที่ ตั้งค่า → ผู้ใช้งาน → สร้างผู้ใช้ใหม่"

## Section 5: Navigation Tips (2 minutes)

**[Scene: Navigate through different sections]**

**Narrator:**
"เคล็ดลับการใช้งานระบบให้มีประสิทธิภาพ"

**Tips:**
1. **การค้นหา:** ใช้ช่องค้นหาด้านบนเพื่อหาข้อมูลอย่างรวดเร็ว
2. **ตัวกรอง:** ใช้ Filter เพื่อแสดงเฉพาะข้อมูลที่ต้องการ
3. **การเรียงลำดับ:** คลิกที่หัวคอลัมน์เพื่อเรียงข้อมูล
4. **การส่งออก:** ส่งออกข้อมูลเป็น Excel หรือ PDF
5. **การพิมพ์:** ใช้ปุ่มพิมพ์ในแต่ละหน้า

**Keyboard Shortcuts:**
- Ctrl + N: สร้างเอกสารใหม่
- Ctrl + S: บันทึก
- Ctrl + P: พิมพ์
