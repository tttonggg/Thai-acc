# Video Tutorial Script: 05 - Financial Reports

## Video Information
- **Title:** Financial Reports (รายงานทางการเงิน)
- **Duration:** 18-20 minutes
- **Target Audience:** Business owners, managers, accountants
- **Prerequisites:** Understanding of journal entries and transactions

## Introduction (1 minute)

**[Scene: Reports dashboard]**

**Narrator:**
"สวัสดีครับ/ค่ะ ในตอนนี้เราจะมาเรียนรู้การอ่านและใช้ประโยชน์จากรายงานทางการเงิน ซึ่งเป็นสิ่งสำคัญในการบริหารธุรกิจ"

**On-screen text:**
- งบดุล (Balance Sheet)
- งบกำไรขาดทุน (Income Statement)
- งบกระแสเงินสด (Cash Flow)

## Section 1: Reports Overview (2 minutes)

**[Scene: Reports menu]**

**Narrator:**
"ระบบมีรายงานทางการเงินครบถ้วนตามมาตรฐานบัญชีไทย"

**Available Reports:**
1. **งบดุล** - แสดงสภาพการเงิน
2. **งบกำไรขาดทุน** - แสดงผลการดำเนินงาน
3. **งบกระแสเงินสด** - แสดงการเคลื่อนไหวเงินสด
4. **งบทดลอง** - รายการบัญชีทั้งหมด
5. **สมุดบัญชีแยกประเภท** - รายละเอียดแต่ละบัญชี
6. **รายงานภาษี** - สำหรับยื่นภาษี

## Section 2: Balance Sheet (4 minutes)

**[Scene: Balance Sheet report]**

**Narrator:**
"งบดุลแสดงสภาพการเงินของบริษัท ณ วันใดวันหนึ่ง"

**Balance Sheet Structure:**
```
งบดุล ณ วันที่ 31 มีนาคม 2026

สินทรัพย์ (Assets)
  สินทรัพย์หมุนเวียน
    เงินสดและเงินฝาก      450,000
    ลูกหนี้การค้า         680,000
    สินค้าคงเหลือ         320,000
    ---------------------------
    รวมสินทรัพย์หมุนเวียน  1,450,000
  
  สินทรัพย์ถาวร
    อาคาร              2,000,000
    เครื่องใช้สำนักงาน     350,000
    หัก ค่าเสื่อมสะสม     (500,000)
    ---------------------------
    รวมสินทรัพย์ถาวร      1,850,000
    
รวมสินทรัพย์              3,300,000

หนี้สิน (Liabilities)
  หนี้สินหมุนเวียน
    เจ้าหนี้การค้า         280,000
    ภาษีขาย              125,000
    เงินกู้ระยะสั้น        300,000
    ---------------------------
    รวมหนี้สินหมุนเวียน     705,000

ทุน (Equity)
  ทุนจดทะเบียน        1,000,000
  กำไรสะสม            1,595,000
  ---------------------------
  รวมทุน               2,595,000

รวมหนี้สินและทุน          3,300,000
```

**Key Ratios:**
- อัตราส่วนสภาพคล่อง = สินทรัพย์หมุนเวียน / หนี้สินหมุนเวียน
- อัตราส่วนทุนหมุนเวียน = สินทรัพย์หมุนเวียน - หนี้สินหมุนเวียน

## Section 3: Income Statement (4 minutes)

**[Scene: Income Statement report]**

**Narrator:**
"งบกำไรขาดทุนแสดงผลการดำเนินงานในช่วงเวลาหนึ่ง"

**Income Statement Structure:**
```
งบกำไรขาดทุน
สำหรับงวด 3 เดือน สิ้นสุดวันที่ 31 มีนาคม 2026

รายได้ (Revenue)
  รายได้ขายสินค้า        2,500,000
  รายได้ค่าบริการ          500,000
  ---------------------------
  รวมรายได้             3,000,000

ต้นทุนขาย (Cost of Goods Sold)
  สินค้าคงเหลือต้นงวด      200,000
  บวก ซื้อสินค้า          1,200,000
  สินค้าที่มีจำหน่าย       1,400,000
  หัก สินค้าคงเหลือปลายงวด (320,000)
  ---------------------------
  ต้นทุนขาย             1,080,000

กำไรขั้นต้น (Gross Profit)  1,920,000

ค่าใช้จ่ายในการขายและบริหาร
  ค่าใช้จ่ายบุคลากร         450,000
  ค่าเช่า                  150,000
  ค่าใช้จ่ายสำนักงาน        120,000
  ค่าเสื่อมราคา             50,000
  ค่าใช้จ่ายอื่นๆ           200,000
  ---------------------------
  รวมค่าใช้จ่าย            970,000

กำไรจากการดำเนินงาน        950,000

รายได้อื่น / ค่าใช้จ่ายอื่น
  ดอกเบี้ยรับ               15,000
  ดอกเบี้ยจ่าย             (30,000)
  ---------------------------
  รวมรายได้อื่นๆ           (15,000)

กำไรสุทธิก่อนภาษี          935,000

ภาษีเงินได้              187,000

กำไรสุทธิ (Net Profit)      748,000
```

**Key Metrics:**
- อัตรากำไรขั้นต้น = กำไรขั้นต้น / รายได้
- อัตรากำไรสุทธิ = กำไรสุทธิ / รายได้

```
  กำไรสุทธิ                748,000
  บวก ค่าเสื่อมราคา         50,000
  บวก/ลบ การเปลี่ยนแปลงหนี้สินหมุนเวียน
    ลูกหนี้การค้า         (180,000)
    เจ้าหนี้การค้า          80,000
    สินค้าคงเหลือ         (120,000)
  ---------------------------
  เงินสดจากการดำเนินงาน     578,000

กิจกรรมลงทุน (Investing)
  ซื้อเครื่องใช้สำนักงาน    (100,000)
  ---------------------------
  เงินสดใช้ในการลงทุน      (100,000)

กิจกรรมจัดหาเงิน (Financing)
  เงินกู้ระยะสั้น          300,000
  ชำระเงินกู้             (200,000)
  ---------------------------
  เงินสดจากการจัดหาเงิน     100,000

เงินสดเพิ่มขึ้น/ลดลง        578,000
เงินสดต้นงวด              100,000
เงินสดปลายงวด             678,000
```

## Section 6: VAT Reports (2 minutes)

**[Scene: VAT report]**

**Narrator:**
"รายงานภาษีมูลค่าเพิ่มสำหรับยื่น ภ.พ.30"

**Output VAT:**
```
ภาษีขาย (Output VAT) - เดือน มีนาคม 2026

เลขที่        วันที่        ลูกค้า           ยอดขาย    VAT 7%
------------------------------------------------------------
INV-001    05/03/2026  บ.เอบีซี        100,000    7,000
INV-002    10/03/2026  บ.ดีอีเอฟ        200,000   14,000
INV-003    15/03/2026  บ.จีเอช          50,000    3,500
------------------------------------------------------------
รวม                              350,000   24,500
```

**Input VAT:**
```
ภาษีซื้อ (Input VAT) - เดือน มีนาคม 2026

เลขที่        วันที่        ผู้ขาย          ยอดซื้อ    VAT 7%
------------------------------------------------------------
PUR-001    02/03/2026  บ.เอ็กซ์         80,000    5,600
PUR-002    12/03/2026  บ.วายซี          30,000    2,100
------------------------------------------------------------
รวม                              110,000    7,700
```

**VAT Summary:**
```
ภาษีขาย (Output VAT):     24,500
ภาษีซื้อ (Input VAT):      7,700
--------------------------------
ภาษีที่ต้องชำระ:          16,800
```
