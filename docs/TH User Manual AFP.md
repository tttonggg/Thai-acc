• ภาพรวมรายได้และรายจ่าย แสดงข้อมูลรายได้ ค่าใช้จ่าย และกําไรทั้งปีของปีปัจจุบัน
เป็นตัวเลขสรุป โดยแสดงข้อมูลเป็นรายเดือนและรายปี
• ด้านรายได้ ข้อมูลในส่วนนี้แสดงข้อมูล 3 สินค้า หรือบริการที่สร้างรายได้ให้มากที่สุด

• ด้านลูกหนี้ แสดงยอดรวมของลูกหนี้ที่เกินกําหนด และกําลังจะถึงกําหนดชําระในเดือนนี้

• ด้านเจ้าหนี้ แสดงยอดรวมของเจ้าหนี้ที่เกินกําหนด และกําลังจะถึงกําหนดชําระในเดือนนี้
และอีกสามเดือนข้างหน้า
• ยอดคงเหลือของเงินสดและเงินฝากธนาคาร ข้อมูลในส่วนนี้แสดงเงินสด
และเงินฝากธนาคารที่มียอดคงเหลือ ณ วันนี้สูงสุด
