# ภาษีหัก ณ ที่จ่าย (Withholding Tax - WHT)
# Withholding Tax Manual

**Section 15 of the Thai Accounting ERP User Manual**

---

## 1. ภาพรวม

ภาษีหัก ณ ที่จ่าย (WHT) คือภาษีที่ผู้จ่ายเงินหักไว้จากจำนวนเงินที่จ่าย แล้วนำส่งกรมสรรพากรแทนผู้รับเงิน

### ผู้ที่ต้องหัก WHT
- บริษัทที่จดทะเบียน VAT
- ห้างหุ้นส่วนที่จดทะเบียน VAT
- นิติบุคคลอื่นที่มีหน้าที่หักภาษี

---

## 2. ประเภทและอัตรา WHT

| ประเภทรายจ่าย | อัตรา | ผู้รับเงิน |
|--------------|-------|------------|
| ค่าเช่า | 5% | บุคคลธรรมดา |
| ค่าโฆษณา | 2% | บุคคลธรรมดา/นิติบุคคล |
| ค่าขนส่ง | 1% | บุคคลธรรมดา/นิติบุคคล |
| ค่าบริการ | 3% | บุคคลธรรมดา |
| ค่าบริการ | 3% | นิติบุคคล |
| เงินปันผล | 10% | ผู้ถือหุ้น |
| ดอกเบี้ย | 15% | บุคคลธรรมดา |
| อื่นๆ | 1-5% | ตามประเภท |

---

## 3. เอกสารที่ต้องออก

### ใบรับรองการหักภาษี (PND 1)
ออกเมื่อ:
- จ่ายเงินให้บุคคลธรรมดา
- หักภาษี ณ ที่จ่าย

### ใบรับรองการหักภาษี (PND 3)
ออกเมื่อ:
- จ่ายเงินให้นิติบุคคล
- หักภาษี ณ ที่จ่าย

### ใบรับรองการหักภาษี (PND 53)
ออกเมื่อ:
- จ่ายเงินให้ต่างประเทศ
- หักภาษี ณ ที่จ่าย

---

## 4. การออกใบรับรอง WHT

**ทางเข้า:** เมนูด้านข้าง → **ภาษี** → **ภาษีหัก ณ ที่จ่าย**

### ขั้นตอน
1. กด **+ สร้างใบรับรอง**
2. เลือกประเภทแบบฟอร์ม (PND 1/3/53)
3. เลือกผู้รับเงิน
4. กรอกจำนวนเงินที่จ่าย
5. ระบบจะคำนวณภาษีหักอัตโนมัติ
6. พิมพ์ใบรับรอง

### ตัวอย่าง: จ่ายค่าเช่า ฿100,000 ให้บุคคลธรรมดา
```
จำนวนเงินที่จ่าย:     ฿100,000
ภาษีหัก ณ ที่จ่าย 5%:    ฿5,000
เงินที่จ่ายสุทธิ:          ฿95,000
```

---

## 5. บันทึกบัญชี WHT

### การจ่ายเงินและหัก WHT
```
เดบิต  ค่าเช่า                    ฿100,000
  เครดิต  เงินสด/ธนาคาร               ฿95,000
  เครดิต  ภาษีหัก ณ ที่จ่ายรอนำส่ง     ฿5,000
```

### เมื่อนำส่งภาษี
```
เดบิต  ภาษีหัก ณ ที่จ่ายรอนำส่ง    ฿5,000
  เครดิต  เงินสด/ธนาคาร                  ฿5,000
```

---

## 6. การยื่นภาษี WHT

### ภาษีหัก ณ ที่จ่าย รายเดือน
- ยื่นแบบออนไลน์ภายในวันที่ 7 ของเดือนถัดไป
- นำส่งเงินภาษีพร้อมแบบแจ้ง

### สถานะการยื่น
| สถานะ | คำอธิบาย |
|--------|---------|
| **ยังไม่ยื่น** | รอยื่นภายในกำหนด |
| **ยื่นแล้ว** | ยื่นแบบออนไลน์แล้ว |
| **นำส่งแล้ว** | จ่ายเงินภาษีแล้ว |

---

## 7. รายงาน WHT

| รายงาน | คำอธิบาย |
|--------|---------|
| **รายงานภาษีหัก ณ ที่จ่าย** | สรุปรายการตามเดือน |
| **รายงานแยกประเภท** | สรุปตามประเภทรายจ่าย |
| **รายงานยื่นภาษี** | รายการที่ยื่นแบบแล้ว |
