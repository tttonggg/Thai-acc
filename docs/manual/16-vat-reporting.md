# ภาษีมูลค่าเพิ่ม (VAT)
# VAT Reporting Manual

**Section 16 of the Thai Accounting ERP User Manual**

---

## 1. ภาพรวม

ภาษีมูลค่าเพิ่ม (VAT) คือภาษีที่จัดเก็บจากการบริโภคสินค้าหรือบริการ ผู้ประกอบการที่จด VAT จะเก็บ VAT จากลูกค้า (VAT ขาย) และชำระ VAT ให้ supplier (VAT ซื้อ) แล้วนำส่งส่วนต่างให้กรมสรรพากร

### อัตรา VAT
- **VAT 7%** — อัตรามาตรฐาน
- **VAT 0%** — สินค้ายกเว้น (อาหารสด, บริการทางการแพทย์, การศึกษา)
- **ยกเว้น** — ไม่มี VAT

---

## 2. VAT ขาย (Output VAT)

### บันทึกบัญชี VAT ขาย
```
เดบิต  ลูกหนี้                  ฿10,700
  เครดิต  รายได้จากการขาย           ฿10,000
  เครดิต  VAT ที่ต้องจ่าย             ฿700
```

### เมื่อนำส่ง VAT
```
เดบิต  VAT ที่ต้องจ่าย            ฿700
  เครดิต  เงินสด/ธนาคาร               ฿700
```

---

## 3. VAT ซื้อ (Input VAT)

### บันทึกบัญชี VAT ซื้อ
```
เดบิต  ค่าสินค้า/บริการ             ฿10,000
เดบิต  VAT ที่ซื้อได้                ฿700
  เครดิต  เจ้าหนี้                    ฿10,700
```

### การขอคืน VAT
เมื่อ VAT ซื้อ > VAT ขาย:
- ยื่นขอคืนส่วนต่างจากกรมสรรพากร
- หรือ Carry Forward ไปเดือนถัดไป

---

## 4. การคำนวณ VAT รายเดือน

### สูตร
```
VAT ที่ต้องนำส่ง = VAT ขาย - VAT ซื้อ (ขอคืนได้)
```

### ตัวอย่าง
```
VAT ขาย รายเดือน:    ฿50,000
VAT ซื้อ รายเดือน:    ฿30,000

VAT ที่ต้องนำส่ง:    ฿20,000
```

---

## 5. การยื่นแบบ VAT

### ภ.พ.30 (แบบแจ้งรายการ VAT)
- ยื่นออนไลน์ภายในวันที่ 23 ของเดือนถัดไป
- นำส่งเงิน VAT พร้อมแบบแจ้ง

### ระยะเวลาการยื่น
| ประเภทผู้ประกอบการ | การยื่น |
|---------------------|---------|
| ยื่นปกติ | รายเดือน |
| ยื่นเฉพาะครึ่งปี | กรณีมีรายได้ไม่เกิน 1.8 ล้านบาท/ครึ่งปี |

---

## 6. สมุดรายวัน VAT

ระบบจัดทำสมุดรายวันอัตโนมัติ:

### สมุดรายวันภาษีขาย
- รายการใบกำกับภาษีขายทั้งหมด
- รวม VAT ขายรายเดือน

### สมุดรายวันภาษีซื้อ
- รายการใบกำกับภาษีซื้อทั้งหมด
- รวม VAT ซื้อรายเดือน

---

## 7. รายงาน VAT

| รายงาน | คำอธิบาย |
|--------|---------|
| **รายงาน VAT ขาย** | แสดงรายละเอียด VAT ขายรายเดือน |
| **รายงาน VAT ซื้อ** | แสดงรายละเอียด VAT ซื้อรายเดือน |
| **รายงานสถานะ VAT** | แสดง VAT คงค้าง รอนำส่ง |
| **รายงานสมุดรายวัน** | สมุดรายวันภาษีขาย/ซื้อ |

---

## 8. การตรวจสอบ VAT

### จุดตรวจสอบ
- ใบกำกับภาษีต้องมีเลขประจำตัวผู้เสียภาษี (13 หลัก)
- วันที่ออกใบกำกับภาษีต้องถูกต้อง
- จำนวน VAT ต้องคำนวณจากฐานภาษี × 7%
- ผู้ขายต้องอยู่ในทะเบียนผู้เสียภาษีที่ถูกต้อง

### ข้อควรระวัง
- VAT ซื้อจากผู้ไม่ได้จด VAT ไม่สามารถขอคืนได้
- ใบกำกับภาษีที่ไม่สมบูรณ์ไม่สามารถใช้ได้
