# ใบเสนอราคา (Quotations)
# Quotations Manual

**Section 7 of the Thai Accounting ERP User Manual**

---

## 1. ภาพรวม

ใบเสนอราคาคือเอกสารที่เสนอราคาสินค้า/บริการให้ลูกค้าพิจารณา ก่อนที่จะสร้างใบแจ้งหนี้จริง

---

## 2. สถานะใบเสนอราคา

| สถานะ | Thai | ความหมาย |
|-------|------|---------|
| **DRAFT** | ฉบับร่าง | สร้างแล้ว ยังไม่ส่ง |
| **SENT** | ส่งแล้ว | ส่งให้ลูกค้าแล้ว |
| **APPROVED** | อนุมัติแล้ว | ลูกค้าตกลงซื้อ |
| **REJECTED** | ปฏิเสธ | ลูกค้าไม่ตกลง |
| **EXPIRED** | หมดอายุ | เกิน 30 วันโดยไม่ตอบ |
| **CONVERTED** | แปลงแล้ว | สร้างใบแจ้งหนี้แล้ว |

---

## 3. การสร้างใบเสนอราคา

**ทางเข้า:** เมนูด้านข้าง → **ขาย** → **ใบเสนอราคา** → **+ สร้างใหม่**

### ขั้นตอน
1. **เลือกลูกค้า**
2. **วันที่** และ **วันหมดอายุ** (ค่าเริ่มต้น +30 วัน)
3. **เพิ่มรายการสินค้า/บริการ**
4. **เลือกเงื่อนไข** — ราคา, VAT, WHT (ถ้ามี)
5. **บันทึก** → เป็น DRAFT หรือ **ส่งเลย** → SENT

---

## 4. การแปลงเป็นใบแจ้งหนี้

เมื่อลูกค้าตกลงซื้อ:
1. เปิดใบเสนอราคาที่ **APPROVED**
2. กดปุ่ม **สร้างใบแจ้งหนี้ (Convert to Invoice)**
3. ระบบจะสร้างใบแจ้งหนี้ใหม่โดยก็อปปี้:
   - ข้อมูลลูกค้า
   - รายการสินค้า/บริการ
   - ราคาและ VAT
4. เปลี่ยนสถานะเป็น **CONVERTED**
