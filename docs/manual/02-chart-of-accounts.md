# Chart of Accounts Manual

**Section 2 of the Thai Accounting ERP User Manual**

---

## 1. ภาพรวมผังบัญชี

ผังบัญชีคือรายการบัญชีทั้งหมดที่ใช้ในระบบ จำนวน **181 บัญชี** ตามมาตรฐาน TFRS และกรมสรรพากรไทย

### โครงสร้างเลขที่บัญชี (Account Codes)

ระบบใช้เลข 4 หลักแรกเป็น **ประเภทบัญชี**:

| ประเภท | รหัส | คำอธิบาย |
|--------|------|---------|
| **สินทรัพย์ (ASSET)** | 1xxx | ทรัพย์สิน ลูกหนี้ เงินสด |
| **หนี้สิน (LIABILITY)** | 2xxx | เจ้าหนี้ สินเชื่อ ภาษีต้องจ่าย |
| **ทุน (EQUITY)** | 3xxx | ทุน กำไรสะสม |
| **รายได้ (REVENUE)** | 4xxx | รายได้จากการขาย บริการ |
| **ค่าใช้จ่าย (EXPENSE)** | 5xxx | ค่าใช้จ่ายในการดำเนินงาน |
| **ต้นทุน (COST)** | 6xxx | ต้นทุนสินค้าขาย |
| **其他 (OTHER)** | 7xxx-9xxx | บัญชีอื่นๆ |

## 2. การจัดการบัญชี

**ทางเข้า:** เมนูด้านข้าง → **บัญชี** → **ผังบัญชี**

> ⚠️ การเพิ่ม/แก้ไข/ลบบัญชีต้องมีสิทธิ์ `ACCOUNTANT` ขึ้นไป

### 2.1 ดูรายการบัญชี
- แสดงรายการบัญชีทั้งหมดในรูปแบบ **ตาราง**
- สามารถ **ค้นหา** ด้วยชื่อบัญชีหรือเลขที่บัญชี
- สามารถ **กรอง** ตามประเภทบัญชี (Asset, Liability, Equity, Revenue, Expense)
- แสดง: เลขที่บัญชี | ชื่อบัญชี | ประเภท | ยอดคงเหลือ

### 2.2 เพิ่มบัญชีใหม่
1. กดปุ่ม **+ เพิ่มบัญชี**
2. กรอกข้อมูล:
   - **เลขที่บัญชี** (4 หลัก) — ต้องไม่ซ้ำกับที่มีอยู่
   - **ชื่อบัญชี (ไทย)**
   - **ชื่อบัญชี (อังกฤษ)** — ถ้ามี
   - **ประเภทบัญชี** — เลือกจาก dropdown
   - **ประเภทย่อย** — เช่น สินทรัพย์หมุนเวียน / สินทรัพย์ไม่หมุนเวียน
3. กด **บันทึก**

### 2.3 แก้ไขบัญชี
1. คลิกที่แถวของบัญชีที่ต้องการแก้ไข
2. กดปุ่ม **แก้ไข** (ไอคอนดินสอ)
3. แก้ไขข้อมูลที่ต้องการ
4. กด **บันทึก**

### 2.4 ลบบัญชี
> ⚠️ **ไม่สามารถลบบัญชีที่มีการใช้งานแล้ว**

หากบัญชียังไม่เคยมีรายการ → กดปุ่ม **ลบ** ได้

## 3. หลักการบัญชีคู่ (Double-Entry)

ทุกรายการบัญชีในระบบต้องมี **เดบิต = เครดิต** เสมอ

### ตัวอย่าง: ขายสินค้าเงินสด ฿10,000 + VAT 7%
```
เดบิต  เงินสด          ฿10,700
  เครดิต  รายได้จากการขาย    ฿10,000
  เครดิต  ภาษีขายที่ต้องจ่าย      ฿700
```

### ตัวอย่าง: ซื้อสินค้าเงินเชื่อ ฿5,000 + VAT 7%
```
เดบิต  ซื้อสินค้า       ฿5,000
เดบิต  ภาษีซื้อ          ฿350
  เครดิต  เจ้าหนี้          ฿5,350
```

## 4. รูปแบบการเก็บข้อมูลเงิน (Satang Format)

**⚠️ สำคัญมาก:** ระบบเก็บจำนวนเงินเป็น **Satang (สตางค์)** ไม่ใช่บาท

| การแสดงผล | การเก็บในฐานข้อมูล |
|-----------|-------------------|
| ฿1,234.56 | 123456 |
| ฿100.00 | 10000 |
| ฿0.50 | 50 |

**หลักการ:**
- ผู้ใช้กรอก: **บาท** (ทศนิยม 2 ตำแหน่ง)
- ระบบแปลง: คูณ 100 → เก็บในฐานข้อมูล
- ระบบแสดง: หาร 100 → แสดงผลเป็นบาท

## 5. บัญชีหลักที่ควรรู้จัก

### สินทรัพย์ (1xxx)
| เลขที่ | ชื่อบัญชี | การใช้งาน |
|--------|---------|---------|
| 1100 | เงินสด | เงินสดในมือ |
| 1200 | ธนาคาร | เงินฝากธนาคาร |
| 1300 | ลูกหนี้ | ลูกค้าค้างชำระ |
| 1400 | สินค้าคงเหลือ | สต็อกสินค้า |
| 1500 | ที่ดิน อาคาร อุปกรณ์ | สินทรัพย์ถาวร |

### หนี้สิน (2xxx)
| เลขที่ | ชื่อบัญชี | การใช้งาน |
|--------|---------|---------|
| 2100 | เจ้าหนี้ | ค่าสินค้า/บริการที่ยังไม่จ่าย |
| 2200 | เงินกู้ | สินเชื่อจากธนาคาร |
| 2300 | ภาษีขายที่ต้องจ่าย | VAT ที่เก็บจากลูกค้าได้ |
| 2400 | ภาษีหัก ณ ที่จ่าย | WHT ที่หักจากผู้รับเหมา |

### รายได้ (4xxx)
| เลขที่ | ชื่อบัญชี | การใช้งาน |
|--------|---------|---------|
| 4100 | รายได้จากการขาย | ขายสินค้า/บริการ |
| 4200 | รายได้อื่น | ดอกเบี้ย เงินปันผล |

### ค่าใช้จ่าย (5xxx)
| เลขที่ | ชื่อบัญชี | การใช้งาน |
|--------|---------|---------|
| 5100 | ต้นทุนสินค้าขาย | COGS |
| 5200 | ค่าใช้จ่ายในการขาย | ค่าขนส่ง ค่าคmission |
| 5300 | ค่าใช้จ่ายในการบริหาร | เงินเดือน ค่าเช่า สาธารณูปโภค |

## 6. การค้นหาและกรองบัญชี

### ค้นหาด่วน
พิมพ์ชื่อบัญชีหรือเลขที่บัญชีในช่อง **ค้นหา** → ระบบกรองทันที

### กรองตามประเภท
ใช้ **Dropdown ประเภทบัญชี** เพื่อแสดงเฉพาะประเภทที่ต้องการ:
- สินทรัพย์
- หนี้สิน
- ทุน
- รายได้
- ค่าใช้จ่าย
- ทั้งหมด
