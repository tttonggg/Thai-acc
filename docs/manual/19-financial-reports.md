# รายงานทางการเงิน (Financial Reports)
# Financial Reports Manual

**Section 19 of the Thai Accounting ERP User Manual**

---

## 1. ภาพรวม

ระบบรายงานทางการเงินช่วยให้:
- สร้างรายงานตามมาตรฐาน TFRS
- วิเคราะห์ผลการดำเนินงาน
- ติดตามสถานะทางการเงิน

---

## 2. รายงานหลัก

### 2.1 งบกระแสเงินสด (Cash Flow Statement)

**ทางเข้า:** เมนูด้านข้าง → **รายงาน** → **งบกระแสเงินสด**

แสดงการเคลื่อนไหวของเงินสดในรอบบัญชี แบ่งเป็น:
- กิจกรรมดำเนินงาน
- กิจกรรมลงทุน
- กิจกรรมจัดหาเงิน

### 2.2 งบกำไรขาดทุน (Income Statement)

**ทางเข้า:** เมนูด้านข้าง → **รายงาน** → **งบกำไรขาดทุน**

แสดงผลการดำเนินงานในรอบบัญชี:
- รายได้
- ค่าใช้จ่าย
- กำไร/ขาดทุน

### 2.3 งบดุล (Balance Sheet)

**ทางเข้า:** เมนูด้านข้าง → **รายงาน** → **งบดุล**

แสดงสถานะทางการเงิน ณ วันที่:
- สินทรัพย์
- หนี้สิน
- ส่วนของผู้ถือหุ้น

---

## 3. รายงานภาษี

### 3.1 ภาษีเงินได้นิติบุคคล

**ทางเข้า:** เมนูด้านข้าง → **รายงาน** → **ภาษีเงินได้**

รายงานสำหรับยื่นภาษีประจำปี:
- รายได้ทั้งปี
- ค่าใช้จ่ายที่รับได้
- กำไรทางภาษี
- ภาษีที่ต้องจ่าย

### 3.2 ภาษีมูลค่าเพิ่ม (VAT)

**ทางเข้า:** เมนูด้านข้าง → **รายงาน** → **VAT**

- สมุดรายวันภาษีขาย
- สมุดรายวันภาษีซื้อ
- ภ.พ.30

### 3.3 ภาษีหัก ณ ที่จ่าย (PND 1/3/53)

**ทางเข้า:** เมนูด้านข้าง → **รายงาน** → **ภาษีหัก ณ ที่จ่าย**

รายงานแยกประเภทตาม PND ที่ต้องยื่น

---

## 4. รายงานบัญชี

### 4.1 รายงานบัญชีแยกประเภท

**ทางเข้า:** เมนูด้านข้าง → **รายงาน** → **บัญชีแยกประเภท**

แสดงรายการเคลื่อนไหวของบัญชีเฉพาะ:
- ยอดยกมา
- รายการในงวด
- ยอดคงเหลือ

### 4.2 รายงานทะเบียนซื้อ-ขาย

**ทางเข้า:** เมนูด้านข้าง → **รายงาน** → **ทะเบียนซื้อ-ขาย**

- ทะเบียนใบรับซื้อ
- ทะเบียนใบแจ้งหนี้
- ทะเบียนใบเสร็จรับเงิน

### 4.3 รายงานลูกหนี้-เจ้าหนี้

**ทางเข้า:** เมนูด้านข้าง → **รายงาน** → **ลูกหนี้/เจ้าหนี้**

- อายุลูกหนี้
- อายุเจ้าหนี้
- รายงานครบกำหนด

---

## 5. การเลือกงวดบัญชี

ทุกรายงานสามารถเลือกช่วงวันที่ได้:

| ตัวเลือก | คำอธิบาย |
|-----------|---------|
| วันนี้ | รายงาน ณ วันปัจจุบัน |
| เดือนนี้ | รายงานเดือนปัจจุบัน |
| ไตรมาสนี้ | รายงานไตรมาสปัจจุบัน |
| ปีนี้ | รายงานปีปัจจุบัน |
| กำหนดเอง | เลือกวันที่เอง |

---

## 6. การส่งออกรายงาน

### รูปแบบที่รองรับ
| รูปแบบ | การใช้งาน |
|---------|----------|
| **PDF** | พิมพ์หรือส่งอีเมล |
| **Excel (XLSX)** | นำไปวิเคราะห์เพิ่มเติม |
| **CSV** | นำเข้าระบบอื่น |

### การส่งออก
1. เปิดรายงานที่ต้องการ
2. เลือกช่วงวันที่
3. กดปุ่ม **ส่งออก**
4. เลือกรูปแบบไฟล์
5. ดาวน์โหลดหรือส่งอีเมล

---

## 7. การตั้งเวลาส่งรายงาน

**ทางเข้า:** เมนูด้านข้าง → **รายงาน** → **ตั้งเวลาส่ง**

สามารถตั้งค่าให้ระบบส่งรายงานอัตโนมัติ:
- รายเดือน → ผู้บริหาร
- รายไตรมาส → ผู้ถือหุ้น
- รายปี → กรมสรรพากร
