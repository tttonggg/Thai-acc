# ใบลดหนี้ / ใบเพิ่มหนี้ (Credit & Debit Notes)
# Credit & Debit Notes Manual

**Section 8 of the Thai Accounting ERP User Manual**

---

## 1. ภาพรวม

เอกสารสำหรับ **ปรับปรุงยอด** จากใบแจ้งหนี้ที่ออกไปแล้ว

| ประเภท | กรณีใช้ |
|--------|---------|
| **ใบลดหนี้ (Credit Note)** | ลูกค้าส่งคืนสินค้า, ลดราคา, ยกเลิก |
| **ใบเพิ่มหนี้ (Debit Note)** | บวกเพิ่มค่าที่ยังไม่เก็บ, ค่าปรับ |

---

## 2. สถานะ

| สถานะ | Thai |
|-------|------|
| **DRAFT** | ฉบับร่าง |
| **ISSUED** | ออกแล้ว |
| **VOID** | ยกเลิก |

---

## 3. การสร้างใบลดหนี้

**ทางเข้า:** เมนูด้านข้าง → **ขาย** → **ใบลดหนี้** → **+ สร้างใหม่**

### ขั้นตอน
1. **เลือกใบแจ้งหนี้ต้นทาง** ที่ต้องการลด/ยกเลิก
2. **เลือกเหตุผล** — ส่งคืน, ลดราคา, ยกเลิก
3. **รายการที่ลด** — เลือกรายการและจำนวนที่ลด
4. **ผ่านรายการ** → ระบบสร้างบันทึกบัญชีอัตโนมัติ

### บันทึกบัญชี (ใบลดหนี้)
```
เดบิต  รายได้จากการขาย (ลด)     ฿1,000
เดบิต  ภาษีขายที่ต้องจ่าย (ลด)      ฿70
  เครดิต  ลูกหนี้ (ลด)             ฿1,070
```

---

## 4. การสร้างใบเพิ่มหนี้

**ทางเข้า:** เมนูด้านข้าง → **ขาย** → **ใบเพิ่มหนี้** → **+ สร้างใหม่**

ใช้เมื่อต้องเก็บค่าเพิ่มจากลูกค้า (เช่น ค่าปรับ, ค่าขนส่งเพิ่มเติม)
