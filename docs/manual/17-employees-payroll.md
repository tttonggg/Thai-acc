# บุคลากรและเงินเดือน (Employees & Payroll)
# Employees & Payroll Manual

**Section 17 of the Thai Accounting ERP User Manual**

---

## 1. ภาพรวม

ระบบบุคลากรและเงินเดือนช่วยให้:
- บันทึกข้อมูลพนักงานครบถ้วน
- คำนวณเงินเดือน ค่าจ้าง ค่าล่วงเวลา
- บันทึกประกันสังคม (SSC) และภาษีเงินได้
- จัดทำรายงานภาษีและประกันสังคม

---

## 2. ข้อมูลพนักงาน

**ทางเข้า:** เมนูด้านข้าง → **บุคลากร** → **พนักงาน**

### ข้อมูลที่ต้องกรอก
| ข้อมูล | คำอธิบาย |
|--------|---------|
| รหัสพนักงาน | รหัสอ้างอิงภายใน |
| ชื่อ-นามสกุล | ชื่อตามบัตรประชาชน |
| ตำแหน่ง | ตำแหน่งงาน |
| แผนก | แผนกที่สังกัด |
| วันที่เข้าทำงาน | วันที่เริ่มงาน |
| ประเภทจ้าง | รายเดือน / รายวัน / จ้างตามสัญญา |
| อัตราเงินเดือน | จำนวนเงินเดือน |
| ประกันสังคม | เลขทะเบียน SSC |
| ภาษีเงินได้ | เลขประจำตัวผู้เสียภาษี |

---

## 3. องค์ประกอบเงินเดือน

| องค์ประกอบ | คำอธิบาย |
|-------------|---------|
| **เงินเดือนปกติ** | ฐานเงินเดือนตามสัญญาจ้าง |
| **ค่าล่วงเวลา (OT)** | ค่าจ้างล่วงเวลา 1.5 เท่า |
| **ค่าครองชีพ** | เงินช่วยค่าครองชีพ (ถ้ามี) |
| **โบนัส** | เงินพิเศษตามผลงาน |
| **ค่าตำแหน่ง** | ค่าตำแหน่งพิเศษ (ถ้ามี) |

---

## 4. การคำนวณเงินเดือน

### ขั้นตอน
1. **เลือกเดือน** ที่ต้องการคำนวณ
2. **เพิ่มรายการ** พนักงานที่ต้องจ่าย
3. **กรอกจำนวนวันทำงาน** หรือ ชั่วโมง OT
4. **ระบบจะคำนวณ** รายได้หัก ดังนี้

### รายการหัก
| รายการ | อัตรา |
|--------|-------|
| ประกันสังคม (SSC) | 5% ของเงินเดือน (สูงสุด ฿750) |
| ภาษีเงินได้ | ตามอัตราภาษีพิเศษ (0.5-5%) |
| ค่าสมาชิกกองทุน | ตามที่หักจริง |

### ตัวอย่าง: เงินเดือน ฿30,000
```
รายได้:                    ฿30,000.00
หัก SSC 5%:                 ฿1,500.00
หักภาษี 0.5%:                ฿150.00
รวมหัก:                    ฿1,650.00
เงินที่ได้รับสุทธิ:          ฿28,350.00
```

---

## 5. การจ่ายเงินเดือน

### ขั้นตอน
1. คำนวณเงินเดือนเสร็จ
2. กดปุ่ม **ผ่านรายการ**
3. ระบบจะสร้างบันทึกบัญชีอัตโนมัติ

### บันทึกบัญชีเงินเดือน
```
เดบิต  ค่าแรงงาน/เงินเดือน      ฿30,000
  เครดิต  ภาษีเงินได้ต้องจ่าย       ฿150
  เครดิต  SSC ต้องจ่าย              ฿1,500
  เครดิต  เงินเดือนต้องจ่าย        ฿28,350
```

---

## 6. ประกันสังคม (Social Security)

### อัตราประกันสังคม
| ส่วน | อัตรา | ผู้รับผิดชอบ |
|------|-------|--------------|
| ลูกจ้าง | 5% | หักจากเงินเดือน |
| นายจ้าง | 5% | บริษัทจ่ายเพิ่ม |
| รวม | 10% | (สูงสุด ฿750/เดือน) |

### เมื่อนำส่ง SSC
```
เดบิต  SSC ต้องจ่าย            ฿1,500
  เครดิต  เงินสด/ธนาคาร            ฿1,500
```

---

## 7. กองทุนสำรองเลี้ยงชีพ (Provident Fund)

### อัตรากองทุน
- สมาชิก: 3-15% ของเงินเดือน
- บริษัท: 3-15% ของเงินเดือน (ตามที่กำหนด)

### การหักและนำส่ง
```
เดบิต  เงินเดือนต้องจ่าย          ฿3,000
  เครดิต  เงินกองทุนสำรองฯ           ฿3,000
```

---

## 8. รายงานบุคลากร

| รายงาน | คำอธิบาย |
|--------|---------|
| **รายงานเงินเดือน** | สรุปรายการจ่ายรายเดือน |
| **รายงาน SSC** | รายงานประกันสังคมรายเดือน |
| **รายงานภาษีเงินได้** | รายงานภาษีรายเดือน/รายปี |
| **รายงานกองทุน** | รายงานกองทุนสำรองเลี้ยงชีพ |
