# ธนาคารและการกระทบยอด (Banking & Reconciliation)
# Banking & Reconciliation Manual

**Section 12 of the Thai Accounting ERP User Manual**

---

## 1. ภาพรวม

ระบบธนาคารและการกระทบยอดช่วยให้:
- จัดการบัญชีธนาคารหลายบัญชี
- ติดตามการเคลื่อนไหวของเงินฝาก
- กระทบยอดระหว่างสมุดบัญชีกับใบแจ้งยอดธนาคาร
- ตรวจจับความผิดปกติของเงินในธนาคาร

---

## 2. บัญชีธนาคาร (Bank Accounts)

### การตั้งค่าบัญชีธนาคาร

**ทางเข้า:** เมนูด้านข้าง → **ธนาคาร** → **บัญชีธนาคาร**

### ข้อมูลที่ต้องกรอก
| ข้อมูล | คำอธิบาย |
|--------|---------|
| ชื่อบัญชี | ชื่อที่ใช้แสดงในระบบ |
| ประเภทบัญชี | ออมทรัพย์ / กระแสรายวัน |
| เลขบัญชี | เลขที่บัญชีธนาคาร |
| ชื่อธนาคาร | ชื่อธนาคาร |
| สาขา | สาขาที่เปิดบัญชี |
| ยอดเริ่มต้น | ยอดเงินฝาก ณ วันที่เริ่มใช้ระบบ |
| บัญชี GL | บัญชีแยกประเภทที่เชื่อมโยง |

---

## 3. การนำเข้าใบแจ้งยอดธนาคาร (Bank Statement Import)

ระบบรองรับการนำเข้าจาก:
- ไฟล์ CSV
- ไฟล์ XLS/XLSX
- การดาวน์โหลดจาก Internet Banking

### ขั้นตอนการนำเข้า
1. เปิดบัญชีธนาคารที่ต้องการ
2. กดปุ่ม **นำเข้าใบแจ้งยอด**
3. เลือกไฟล์จากคอมพิวเตอร์
4. ระบบจะ parse ข้อมูลและแสดงตัวอย่าง
5. ตรวจสอบความถูกต้อง → ยืนยันการนำเข้า

### การจับคู่รายการอัตโนมัติ (Auto-Matching)

ระบบจะจับคู่รายการจากใบแจ้งยอดกับ:
- ใบเสร็จรับเงิน (Receipts)
- การชำระเงิน (Payments)
- รายการปรับปรุงอื่นๆ

---

## 4. การกระทบยอด (Reconciliation)

### ขั้นตอนการกระทบยอด
1. เลือกบัญชีธนาคารและงวดที่ต้องการกระทบยอด
2. ระบบจะแสดงรายการที่ยังไม่กระทบยอด
3. ตรวจสอบรายการแต่ละรายการ:
   - ✅ ถ้ารายการตรงกับใบแจ้งยอด → กด **กระทบยอด**
   - ❌ ถ้ารายการไม่ตรง → ตรวจสอบเพิ่มเติม
4. บันทึกผลการกระทบยอด

### รายการที่ต้องกระทบยอด
| ประเภท | คำอธิบาย |
|--------|---------|
| **เดบิต** | รายการขาเข้า (เงินเข้า) |
| **เครดิต** | รายการขาออก (เงินออก) |
| **คงค้าง** | รายการที่ยังไม่ได้กระทบยอด |

---

## 5. รายงานธนาคาร

### รายงานที่มี
- **รายงานกระทบยอด** — แสดงความต่างระหว่างสมุดบัญชีกับใบแจ้งยอด
- **รายงานการเคลื่อนไหวบัญชีธนาคาร** — แสดงรายละเอียดรายการทั้งหมด
- **รายงานเช็คค้าง** — แสดงเช็คที่ยังไม่ขึ้นเงิน

---

## 6. การจัดการเช็ค (Cheque Management)

### สถานะเช็ค

| สถานะ | คำอธิบาย |
|--------|---------|
| **ออกแล้ว** | ออกเช็คให้ผู้จัดจำหน่ายแล้ว |
| **ขึ้นเงินแล้ว** | เช็คผ่านธนาคารแล้ว |
| **ยกเลิก** | เช็คถูกยกเลิก |
| **ค้าง** | เช็คยังไม่ขึ้นเงิน (เกินกำหนด) |

### การติดตามเช็คค้าง
1. เปิดรายงาน **เช็คค้าง**
2. ระบบจะแสดงเช็คที่ยังไม่ขึ้นเงินเกิน 30 วัน
3. สามารถกด **รายงาน** เพื่อติดตาม

---

## 7. บันทึกบัญชีอัตโนมัติ

เมื่อนำเข้าใบแจ้งยอดและกระทบยอดแล้ว ระบบจะสร้างบันทึกบัญชีอัตโนมัติ:

### กรณีเงินเข้าธนาคาร
```
เดบิต  ธนาคาร            ฿10,000
  เครดิต  ลูกหนี้ / รายได้      ฿10,000
```

### กรณีเงินออกจากธนาคาร
```
เดบิต  เจ้าหนี้ / ค่าใช้จ่าย    ฿5,000
  เครดิต  ธนาคาร                ฿5,000
```
