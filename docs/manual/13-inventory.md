# การจัดการสินค้าคงเหลือ (Inventory Management)
# Inventory Management Manual

**Section 13 of the Thai Accounting ERP User Manual**

---

## 1. ภาพรวม

ระบบสินค้าคงเหลือช่วยให้:
- ติดตามสินค้าคงเหลือในคลัง
- บันทึกการรับ-จ่ายสินค้า
- คำนวณต้นทุนสินค้าโดยอัตโนมัติ
- ตรวจนับสินค้าประจำปี

---

## 2. การตั้งค่าสินค้า (Products Setup)

**ทางเข้า:** เมนูด้านข้าง → **สินค้า** → **รายการสินค้า**

### ข้อมูลสินค้า
| ข้อมูล | คำอธิบาย |
|--------|---------|
| รหัสสินค้า | รหัสอ้างอิงภายใน |
| ชื่อสินค้า | ชื่อที่ใช้แสดง |
| หมวดสินค้า | กลุ่มสินค้า |
| หน่วยนับ | ชิ้น, กิโลกรัม, ลิตร ฯลฯ |
| ราคาซื้อ | ราคาต้นทุน |
| ราคาขาย | ราคาขายปกติ |
| บัญชีสินค้า | บัญชีแยกประเภทที่ใช้ |

---

## 3. คลังสินค้า (Warehouses)

**ทางเข้า:** เมนูด้านข้าง → **สินค้า** → **คลังสินค้า**

ระบบรองรับหลายคลัง:
- สินค้าสำนักงานใหญ่
- สินค้าสาขา
- สินค้าตัวแทน

---

## 4. การรับสินค้า (Stock Receipt)

### การรับสินค้าจากใบสั่งซื้อ
1. เปิดใบสั่งซื้อที่ **APPROVED**
2. กดปุ่ม **รับสินค้า (Receive)**
3. บันทึกจำนวนที่ได้รับจริง
4. ระบบจะอัปเดตยอดสินค้าคงเหลือ

### การรับสินค้าแบบไม่มีใบสั่งซื้อ
1. เมนู → **สินค้า** → **รับสินค้า**
2. เลือกผู้จัดจำหน่าย
3. เพิ่มรายการสินค้าและจำนวน
4. บันทึก

---

## 5. การจ่ายสินค้า (Stock Issue)

### การจ่ายสินค้าตามใบสั่งซื้อ
1. เปิดใบรับซื้อที่ผ่านรายการแล้ว
2. กดปุ่ม **จ่ายสินค้า (Issue)**
3. ระบบจะลดยอดสินค้าคงเหลือ

---

## 6. การโอนสินค้าระหว่างคลัง (Stock Transfer)

1. เมนู → **สินค้า** → **โอนสินค้า**
2. เลือกสินค้าที่ต้องการโอน
3. ระบบจะลดยอดคลังต้นทางและเพิ่มยอดคลังปลายทาง

---

## 7. การตรวจนับสินค้า (Stock Take)

### ขั้นตอนการตรวจนับ
1. **เปิดใบตรวจนับ** — เมนู → **สินค้า** → **ตรวจนับ**
2. **เลือกคลัง** ที่ต้องการตรวจนับ
3. **นับจำนวนจริง** และบันทึกลงในระบบ
4. **อนุมัติ** → ระบบจะปรับยอดสินค้าคงเหลือตามจำนวนจริง

### บันทึกบัญชีอัตโนมัติ (เมื่อตรวจนับ)
```
# กรณีสินค้าเกิน
เดบิต  สินค้าคงเหลือ          ฿1,000
  เครดิต  กำไรจากการตรวจนับ       ฿1,000

# กรณีสินค้าขาด
เดบิต  ขาดทุนจากการตรวจนับ    ฿500
  เครดิต  สินค้าคงเหลือ            ฿500
```

---

## 8. การคำนวณต้นทุน (Cost Calculation)

ระบบรองรับวิธีคำนวณต้นทุนหลายแบบ:

| วิธี | คำอธิบาย |
|------|---------|
| **FIFO** | เข้าก่อน ออกก่อน |
| **Average** | ต้นทุนเฉลี่ย |
| **Standard** | ต้นทุนมาตรฐาน |

---

## 9. รายงานสินค้าคงเหลือ

| รายงาน | คำอธิบาย |
|--------|---------|
| **รายงานยอดสินค้าคงเหลือ** | แสดงจำนวนและมูลค่าสินค้า |
| **รายงานการเคลื่อนไหวสินค้า** | แสดงรับ-จ่าย-คงเหลือ |
| **รายงานสินค้าใกล้หมด** | แจ้งเมื่อสินค้าต่ำกว่าขั้นต่ำ |
| **รายงานสินค้าค้าง** | สินค้าที่ไม่ขายนานเกินไป |
