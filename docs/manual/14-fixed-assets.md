# สินทรัพย์ถาวร (Fixed Assets)
# Fixed Assets Manual

**Section 14 of the Thai Accounting ERP User Manual**

---

## 1. ภาพรวม

ระบบสินทรัพย์ถาวรช่วยให้:
- บันทึกข้อมูลสินทรัพย์ครบถ้วน
- คำนวณค่าเสื่อมราคาอัตโนมัติ
- ติดตามการซ่อมและบำรุงรักษา
- จัดทำรายงานภาษีทรัพย์สิน

---

## 2. ประเภทสินทรัพย์ถาวร

| ประเภท | อายุการใช้งาน | อัตราค่าเสื่อม |
|--------|--------------|---------------|
| ที่ดิน | ไม่คิดค่าเสื่อม | — |
| อาคาร | 20 ปี | 5% |
| ครุภัณฑ์ | 5 ปี | 20% |
| ยานพาหนะ | 5 ปี | 20% |
| เครื่องจักร | 5-10 ปี | 10-20% |
| สินทรัพย์ไม่มีตัวตน | 10 ปี | 10% |

---

## 3. การลงทะเบียนสินทรัพย์ใหม่

**ทางเข้า:** เมนูด้านข้าง → **สินทรัพย์** → **ลงทะเบียนสินทรัพย์**

### ข้อมูลที่ต้องกรอก
| ข้อมูล | คำอธิบาย |
|--------|---------|
| รหัสสินทรัพย์ | รหัสอ้างอิงภายใน |
| ชื่อสินทรัพย์ | ชื่อที่ใช้แสดง |
| ประเภทสินทรัพย์ | ประเภทตามตาราง |
| วันที่ได้มา | วันที่ซื้อหรือได้รับ |
| ราคาที่ได้มา | ราคาซื้อหรือต้นทุน |
| อายุการใช้งาน | จำนวนปี |
| มูลค่าซาก | มูลค่าคงเหลือเมื่อหมดอายุ |
| บัญชีสินทรัพย์ | บัญชีแยกประเภท |
| บัญชีค่าเสื่อม | บัญชีค่าเสื่อมสะสม |

---

## 4. การคำนวณค่าเสื่อมราคา (Depreciation)

ระบบคำนวณค่าเสื่อมราคาอัตโนมัติทุกเดือน:

### วิธีคำนวณ
```
ค่าเสื่อมราคารายเดือน = (ราคาที่ได้มา - มูลค่าซาก) ÷ อายุการใช้งาน (เดือน)
```

### ตัวอย่าง: ซื้อเครื่องปรับอากาศ ฿50,000
```
ราคาที่ได้มา:      ฿50,000
มูลค่าซาก:           ฿5,000
อายุการใช้งาน:  5 ปี (60 เดือน)

ค่าเสื่อมต่อเดือน = (50,000 - 5,000) ÷ 60
                  = ฿750/เดือน
```

---

## 5. บันทึกบัญชีค่าเสื่อมราคา

ระบบจะสร้างบันทึกบัญชีอัตโนมัติเมื่อปิดงวดบัญชี:

```
เดบิต  ค่าเสื่อมราคา (ค่าใช้จ่าย)    ฿750
  เครดิต  ค่าเสื่อมสะสม (สินทรัพย์)      ฿750
```

---

## 6. การขายหรือจำหน่ายสินทรัพย์

### ขั้นตอน
1. เปิดสินทรัพย์ที่ต้องการขาย
2. กดปุ่ม **จำหน่าย**
3. กรอกราคาขายและวันที่
4. ระบบจะคำนวณกำไร/ขาดทุนจากการจำหน่าย

### บันทึกบัญชี (เมื่อขาย)
```
# กรณีกำไร
เดบิต  เงินสด/ธนาคาร              ฿30,000
เดบิต  ค่าเสื่อมสะสม               ฿15,000
  เครดิต  สินทรัพย์                   ฿40,000
  เครดิต  กำไรจากการจำหน่ายสินทรัพย์   ฿5,000
```

---

## 7. การประเมินราคาสินทรัพย์ใหม่ (Revaluation)

เมื่อต้องการประเมินราคาสินทรัพย์ใหม่:
1. เปิดสินทรัพย์ที่ต้องการประเมิน
2. กดปุ่ม **ประเมินราคาใหม่**
3. กรอกราคาประเมินใหม่
4. ระบบจะบันทึกผลต่างจากการประเมิน

---

## 8. การซ่อมและบำรุงรักษา

บันทึกการซ่อมแต่ละครั้ง:
1. เปิดสินทรัพย์
2. กด **เพิ่มการซ่อม**
3. กรอกรายละเอียดและค่าใช้จ่าย

---

## 9. รายงานสินทรัพย์

| รายงาน | คำอธิบาย |
|--------|---------|
| **ทะเบียนสินทรัพย์** | แสดงรายละเอียดทุกสินทรัพย์ |
| **รายงานค่าเสื่อมราคา** | แสดงค่าเสื่อมรายเดือน/รายปี |
| **รายงานภาษีทรัพย์สิน** | รายงานสำหรับยื่นภาษี |
| **รายงานการซ่อม** | สรุปค่าใช้จ่ายในการซ่อม |
