# การลา (Leave Management)
# Leave Management Manual

**Section 18 of the Thai Accounting ERP User Manual**

---

## 1. ภาพรวม

ระบบการลาช่วยให้:
- บันทึกการลาของพนักงาน
- ติดตามสิทธิ์การลาคงเหลือ
- อนุมัติหรือปฏิเสธคำขอลา
- สร้างรายงานสรุปการลา

---

## 2. ประเภทการลา

| ประเภท | วันต่อปี | คำอธิบาย |
|---------|----------|----------|
| **ลาป่วย** | 30 วัน | ลาพักเนื่องจากเจ็บป่วย |
| **ลากิจ** | 3-10 วัน | ลาทำธุระส่วนตัว |
| **ลาพักผ่อนประจำปี** | 6-15 วัน | ลาพักผ่อนตามสิทธิ์ |
| **ลาคลอด** | 90 วัน | ลาสำหรับมารดาคลอดบุตร |
| **ลาบวช** | 15-30 วัน | ลาสำหรับการบวชพระ |
| **ลาอื่นๆ** | ตามที่กำหนด | ลาศึกษา, ลาทหาร ฯลฯ |

---

## 3. การขออนุญาตลา

**ทางเข้า:** เมนูด้านข้าง → **บุคลากร** → **การลา** → **+ ขอลาใหม่**

### ขั้นตอน
1. **เลือกประเภทการลา**
2. **กรอกวันที่เริ่มลา** และ **วันสิ้นสุด**
3. **ระบบจะแสดง** จำนวนวันลาและสิทธิ์คงเหลือ
4. **กรอกเหตุผล** (ถ้าต้องการ)
5. **ส่งคำขอ** → รอผู้อนุมัติ

### สถานะคำขอลา
| สถานะ | คำอธิบาย |
|--------|---------|
| **รออนุมัติ** | ส่งคำขอแล้ว รอผู้บังคับบัญชาอนุมัติ |
| **อนุมัติแล้ว** | ผู้บังคับบัญชาอนุมัติแล้ว |
| **ปฏิเสธ** | ผู้บังคับบัญชาปฏิเสธ |
| **ยกเลิก** | ผู้ขอลายกเลิกคำขอ |

---

## 4. การอนุมัติการลา

**ทางเข้า:** เมนูด้านข้าง → **บุคลากร** → **อนุมัติการลา**

### ขั้นตอน
1. ดูรายการคำขอลาที่รออนุมัติ
2. ตรวจสอบข้อมูลและสิทธิ์การลา
3. กด **อนุมัติ** หรือ **ปฏิเสธ**
4. ระบบจะอัปเดตสิทธิ์การลาคงเหลือ

---

## 5. การตรวจสอบสิทธิ์การลา

ดูสิทธิ์การลาของพนักงาน:
1. เมนู → **บุคลากร** → **พนักงาน**
2. เลือกพนักงาน
3. แท็บ **สิทธิ์การลา**
4. ระบบจะแสดง:
   - สิทธิ์ประจำปี
   - ลาแล้ว
   - คงเหลือ

### ตัวอย่าง
```
ลาพักผ่อนประจำปี:
- สิทธิ์ปีนี้:   10 วัน
- ลาแล้ว:         3 วัน
- คงเหลือ:        7 วัน
```

---

## 6. บันทึกบัญชีการลา

โดยปกติการลาไม่ต้องบันทึกบัญชี แต่กรณี:
- ลาที่มีค่าจ้าง → บันทึกเป็นค่าใช้จ่าย
- ลาที่ไม่มีค่าจ้าง → ไม่บันทึกบัญชี

---

## 7. รายงานการลา

| รายงาน | คำอธิบาย |
|--------|---------|
| **สรุปการลารายเดือน** | แสดงการลาทั้งหมดในเดือน |
| **สิทธิ์การลาคงเหลือ** | แสดงสิทธิ์ของพนักงานทั้งหมด |
| **รายงานลาป่วยสะสม** | ติดตามลาป่วยเกินกำหนด |
