# การจัดการระบบ (System Administration)
# System Administration Manual

**Section 20 of the Thai Accounting ERP User Manual**

---

## 1. ภาพรวม

ระบบการจัดการระบบช่วยให้:
- ตั้งค่าข้อมูลธุรกิจ
- จัดการผู้ใช้งาน
- กำหนดสิทธิ์การใช้งาน
- สำรองข้อมูล

---

## 2. ข้อมูลธุรกิจ (Business Profile)

**ทางเข้า:** เมนูด้านข้าง → **ตั้งค่า** → **ข้อมูลธุรกิจ**

### ข้อมูลที่ต้องกรอก
| ข้อมูล | คำอธิบาย |
|---------|---------|
| ชื่อบริษัท | ชื่อนิติบุคคลตามทะเบียน |
| เลขประจำตัวผู้เสียภาษี | 13 หลัก |
| ที่อยู่ | ที่อยู่ตามทะเบียน |
| โทรศัพท์ | เบอร์ติดต่อ |
| อีเมล | อีเมลบริษัท |
| เลขทะเบียนบริษัท | ทะเบียนการค้า |
| ภาษีท้องถิ่น | รหัสภาษีท้องถิ่น |

---

## 3. การจัดการผู้ใช้งาน (User Management)

**ทางเข้า:** เมนูด้านข้าง → **ตั้งค่า** → **ผู้ใช้งาน**

### การเพิ่มผู้ใช้ใหม่
1. กด **+ เพิ่มผู้ใช้**
2. กรอกอีเมลและชื่อ
3. กำหนดบทบาท (Role)
4. เชื่อมโยงกับพนักงาน (ถ้ามี)
5. ส่งอีเมลเชิญ

### การรีเซ็ตรหัสผ่าน
1. เลือกผู้ใช้
2. กด **รีเซ็ตรหัสผ่าน**
3. ระบบจะส่งลิงก์ตั้งรหัสผ่านใหม่ทางอีเมล

---

## 4. การจัดการบทบาท (Role Management)

**ทางเข้า:** เมนูด้านข้าง → **ตั้งค่า** → **บทบาท**

### บทบาทเริ่มต้น
| บทบาท | สิทธิ์ |
|--------|--------|
| **OWNER** | เจ้าของ — เข้าถึงทุกอย่าง รวมถึงจัดการบทบาท |
| **ADMIN** | ผู้ดูแล — จัดการผู้ใช้และการตั้งค่า |
| **SENIOR_ACCOUNTANT** | บัญชีอาวุโส — อนุมัติรายการ |
| **ACCOUNTANT** | บัญชี — สร้างเอกสาร |
| **USER** | ผู้ใช้ทั่วไป — ดูรายงาน |
| **VIEWER** | ผู้ชม — ดูอย่างเดียว |

### การกำหนดสิทธิ์เอง
1. เลือกบทบาทที่ต้องการแก้ไข
2. เลือก **โมดูล** ที่ต้องการให้เข้าถึง
3. กำหนด **การกระทำ** ที่อนุญาต:
   - ดู (View)
   - สร้าง (Create)
   - แก้ไข (Edit)
   - ลบ (Delete)
   - อนุมัติ (Approve)
4. บันทึกการเปลี่ยนแปลง

---

## 5. การตั้งค่าบัญชี (Accounting Settings)

**ทางเข้า:** เมนูด้านข้าง → **ตั้งค่า** → **บัญชี**

### รายการตั้งค่า
| ตั้งค่า | คำอธิบาย |
|---------|---------|
| สิ้นสุดปีบัญชี | เดือนสิ้นสุดปีบัญชี |
| สกุลเงิน | THB (บาท) |
| จุดทศนิยม | 2 ตำแหน่ง |
| อัตรา VAT | 7% |
| อัตรา WHT มาตรฐาน | ค่าเริ่มต้น |

---

## 6. การสำรองข้อมูล (Backup)

**ทางเข้า:** เมนูด้านข้าง → **ตั้งค่า** → **สำรองข้อมูล**

### การสำรองด้วยตนเอง
1. กดปุ่ม **สำรองข้อมูลตอนนี้**
2. ระบบจะสร้างไฟล์ ZIP
3. ดาวน์โหลดไฟล์ไปเก็บรักษา

### การกู้คืนข้อมูล
1. กดปุ่ม **กู้คืนข้อมูล**
2. เลือกไฟล์สำรอง
3. ระบบจะแจ้งเตือนก่อน
4. ยืนยันการกู้คืน

### การตั้งค่าสำรองอัตโนมัติ
- สำรองทุกวัน → เก็บ 7 วันล่าสุด
- สำรองทุกสัปดาห์ → เก็บ 4 สัปดาห์ล่าสุด

---

## 7. บันทึกการตรวจสอบ (Audit Log)

**ทางเข้า:** เมนูด้านข้าง → **ตั้งค่า** → **บันทึกตรวจสอบ**

ระบบจะบันทึกทุกการกระทำ:
| การกระทำ | รายละเอียด |
|----------|-------------|
| เข้าสู่ระบบ | วันเวลา IP |
| สร้างเอกสาร | ประเภท เลขที่ |
| แก้ไขเอกสาร | ก่อน-หลัง |
| อนุมัติเอกสาร | ผู้อนุมัติ |
| ลบข้อมูล | ผู้ลบ วันเวลา |
| แก้ไขสิทธิ์ | ผู้แก้ไข |

---

## 8. การตั้งค่าอีเมล (Email Settings)

**ทางเข้า:** เมนูด้านข้าง → **ตั้งค่า** → **อีเมล**

### การตั้งค่า SMTP
| รายการ | คำอธิบาย |
|---------|---------|
| SMTP Server | ที่อยู่เซิร์ฟเวอร์ |
| Port | พอร์ต (587 สำหรับ TLS) |
| Username | ชื่อผู้ใช้ |
| Password | รหัสผ่าน |
| อีเมลผู้ส่ง | อีเมลที่ใช้ส่ง |

### การทดสอบ
กด **ส่งอีเมลทดสอบ** เพื่อตรวจสอบความถูกต้อง

---

## 9. การออกจากระบบทุกอุปกรณ์

เมื่อต้องการออกจากระบบทุกอุปกรณ์ที่เคยเข้าใช้:
1. ไปที่ **โปรไฟล์** → **ความปลอดภัย**
2. กด **ออกจากระบบทุกอุปกรณ์**
3. ยืนยันการดำเนินการ

---

## 10. การแจ้งเตือน (Notifications)

**ทางเข้า:** เมนูด้านข้าง → **ตั้งค่า** → **แจ้งเตือน**

### ช่องทางแจ้งเตือน
| ช่องทาง | คำอธิบาย |
|---------|----------|
| อีเมล | ส่งอีเมลเมื่อมีเหตุการณ์สำคัญ |
| ภายในระบบ | แจ้งเตือนในหน้าเว็บ |
| LINE | ส่งข้อความไป LINE (ถ้าตั้งค่า) |

### เหตุการณ์ที่แจ้งเตือน
- คำขอลารออนุมัติ
- ใบแจ้งหนี้ครบกำหนด
- สินค้าคงเหลือต่ำ
- รายงานตามเวลาที่ตั้ง
