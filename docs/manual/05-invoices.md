# ใบแจ้งหนี้ / ใบกำกับภาษี (Invoices)
# Invoices Manual

**Section 5 of the Thai Accounting ERP User Manual**

---

## 1. ภาพรวม

ใบแจ้งหนี้คือเอกสารทางการค้าที่ใช้ **เรียกเก็บเงินจากลูกค้า** ระบบรองรับทั้ง:
- **ใบแจ้งหนี้ (Invoice)** — สำหรับขายสินค้า/บริการทั่วไป
- **ใบกำกับภาษี (Tax Invoice)** — สำหรับขายที่มี VAT

ตามกฎหมายไทย ธุรกิจที่จด VAT ต้องออกใบกำกับภาษีทุกครั้งที่ขายสินค้าหรือบริการ

---

## 2. สถานะใบแจ้งหนี้ (Invoice Status)

| สถานะ | Thai | ความหมาย |
|-------|------|---------|
| **DRAFT** | ฉบับร่าง | สร้างแล้ว ยังไม่ผ่านรายการ |
| **ISSUED** | ออกแล้ว | ผ่านรายการแล้ว รอชำระเงิน |
| **PAID** | ชำระแล้ว | ลูกค้าชำระเงินครบแล้ว |
| **PARTIAL** | ชำระบางส่วน | ชำระบางส่วน |
| **VOID** | ยกเลิก | ถูกยกเลิก (ต้องออกใบลดหนี้แทน) |

---

## 3. การสร้างใบแจ้งหนี้ใหม่

**ทางเข้า:** เมนูด้านข้าง → **ขาย** → **ใบแจ้งหนี้** → **+ สร้างใหม่**

### ขั้นตอน
1. **เลือกลูกค้า** — ค้นหาจากรายการลูกค้า
2. **วันที่** — วันที่ออกใบแจ้งหนี้
3. **เลขที่** — ระบบจะสร้างอัตโนมัติ (INV-2569-XXXX)
4. **รายการสินค้า/บริการ** — เพิ่มรายการ:
   - ชื่อสินค้า/บริการ
   - จำนวน
   - ราคาต่อหน่วย (ยังไม่รวม VAT)
5. **VAT** — เลือกอัตรา (7% หรือ ยกเว้น)
6. **หัก ณ ที่จ่าย (WHT)** — ถ้ามี เลือกประเภท WHT

### ตัวอย่าง: ขายสินค้า ฿10,000 + VAT 7%
```
รายการ:
- สินค้าชิ้นที่ 1      ฿10,000

รวมเอกสาร:           ฿10,000
VAT 7%:                ฿700
รวมทั้งสิ้น:          ฿10,700
```

---

## 4. การผ่านรายการ (Issue/Post)

เมื่อสร้างใบแจ้งหนี้แล้ว ต้อง **ผ่านรายการ** เพื่อ:
1. อัปเดตยอดลูกหนี้
2. สร้าง **บันทึกบัญชีอัตโนมัติ**
3. หัก WHT (ถ้ามี) ไปยังบัญชีภาษีหัก ณ ที่จ่าย

### บันทึกบัญชีอัตโนมัติ (เมื่อ Issue)
```
เดบิต  ลูกหนี้            ฿10,700
  เครดิต  รายได้จากการขาย    ฿10,000
  เครดิต  ภาษีขายที่ต้องจ่าย      ฿700
```

---

## 5. การพิมพ์ PDF

### ปุ่มพิมพ์ใบแจ้งหนี้
1. เปิดใบแจ้งหนี้ที่ต้องการ
2. กดปุ่ม **พิมพ์ PDF**
3. ระบบจะสร้าง PDF พร้อมข้อมูลครบถ้วน:
   - ข้อมูลธุรกิจ
   - ข้อมูลลูกค้า
   - รายการสินค้า/บริการ
   - VAT และ WHT
   - QR Code สำหรับชำระเงิน

### องค์ประกอบที่ต้องมีตามกฎหมาย
- ชื่อและที่อยู่ผู้ขาย
- ชื่อและที่อยู่ผู้ซื้อ
- เลขประจำตัวผู้เสียภาษี
- วันที่ออก
- เลขที่ใบแจ้งหนี้
- รายละเอียดสินค้า/บริการ (ชื่อ จำนวน ราคา)
- จำนวนภาษีมูลค่า
- ลายมือชื่อผู้มีอำนาจ

---

## 6. การยกเลิกใบแจ้งหนี้ (Void)

> ⚠️ ห้ามลบใบแจ้งหนี้ที่ออกแล้ว ต้องใช้ **ใบลดหนี้** แทน

### ขั้นตอนการยกเลิก
1. เปิดใบแจ้งหนี้ที่ต้องการยกเลิก
2. กดปุ่ม **ยกเลิก**
3. ระบบจะเปลี่ยนสถานะเป็น **VOID**
4. ต้องสร้าง **ใบลดหนี้ (Credit Note)** เพื่อปรับยอดลูกหนี้

### กรณีที่ต้องยกเลิก
- ข้อมูลลูกค้าผิด
- รายการสินค้าผิด
- ราคาผิด
- ยกเลิกการขายทั้งหมด

---

## 7. การเก็บภาษี ณ ที่จ่าย (WHT)

ระบบรองรับ WHT สำหรับการจ่ายเงินให้ผู้รับเหมาชาวต่างชาติหรือบุคคลที่มีหน้าที่หักภาษี

### WHT ประเภทที่รองรับ
| ประเภท | อัตรา | กรณีใช้ |
|--------|-------|---------|
| ค่าเช่า | 5% | ค่าเช่าทรัพย์สิน |
| ค่าโฆษณา | 2% | ค่าโฆษณา |
| ค่าขนส่ง | 1% | ค่าขนส่งสินค้า |
| ค่าบริการ | 3% | ค่าบริการต่างๆ |
| อื่นๆ | 1-5% | ตามประเภทรายจ่าย |

### การหัก WHT อัตโนมัติ
```
เดบิต  ค่าบริการ                ฿10,000
  เครดิต  เจ้าหนี้              ฿9,700
  เครดิต  ภาษีหัก ณ ที่จ่าย (WHT)     ฿300
```

---

## 8. การอ้างอิงใบแจ้งหนี้

ใบแจ้งหนี้สามารถอ้างอิงเอกสารอื่นได้:
- **ใบเสนอราคา (Quotation)** — อ้างอิงจากใบเสนอราคาที่อนุมัติ
- **ใบสั่งซื้อ (Purchase Order)** — อ้างอิงใบสั่งซื้อจากลูกค้า
- **สัญญา** — อ้างอิงเลขที่สัญญา
- **ใบรับสินค้า (GRN)** — อ้างอิงการรับสินค้า
