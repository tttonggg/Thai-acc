# การแก้ไขปัญหาบันทึกการเคลื่อนไหวสินค้า

## ปัญหา
ไม่สามารถบันทึกการเคลื่อนไหวสินค้า (Stock Movement) ได้

## สาเหตุ
ไม่มีคลังสินค้า (Warehouse) ในระบบ การบันทึกการเคลื่อนไหวสินค้าจำเป็นต้องมีคลังสินค้าก่อน

## การแก้ไข

### 1. สร้างคลังสินค้าเริ่มต้น
```bash
curl -X POST http://localhost:3000/api/warehouses \
  -H "Content-Type: application/json" \
  -d '{
    "code": "MAIN",
    "name": "คลังสินค้าหลัก",
    "type": "MAIN",
    "location": "สำนักงานใหญ่"
  }'
```

**ผลลัพธ์:** ✅ สร้างคลังสินค้าสำเร็จ
- ID: `cmmua0928009nrq6b7t5pvkys`
- ชื่อ: คลังสินค้าหลัก
- รหัส: MAIN

### 2. ทดสอบบันทึกการเคลื่อนไหวสินค้า
```bash
curl -X POST http://localhost:3000/api/stock-movements \
  -H "Content-Type: application/json" \
  -d '{
    "productId": "cmmnqsfo1002j03ypg1mqpq7i",
    "warehouseId": "cmmua0928009nrq6b7t5pvkys",
    "type": "RECEIVE",
    "quantity": 100,
    "unitCost": 50.00,
    "notes": "รับสินค้าเข้าคลัง"
  }'
```

**ผลลัพธ์:** ✅ บันทึกสำเร็จ
- รับสินค้า: 100 หน่วย
- ต้นทุน: 50 บาท/หน่วย
- มูลค่ารวม: 5,000 บาท

## สถานะปัจจุบัน

### คลังสินค้า
| รหัส | ชื่อ | สถานะ |
|-----|------|--------|
| MAIN | คลังสินค้าหลัก | ✅ Active |

### การเคลื่อนไหวสินค้า
| ประเภท | สินค้า | จำนวน | มูลค่า |
|--------|--------|-------|--------|
| RECEIVE | สินค้าตัวอย่าง A | 100 | 5,000 บาท |

### ยอดคงเหลือสินค้า
| สินค้า | จำนวน | ต้นทุน/หน่วย | มูลค่ารวม |
|--------|-------|-------------|-----------|
| สินค้าตัวอย่าง A | 100 | 50 บาท | 5,000 บาท |

## ประเภทการเคลื่อนไหวสินค้า
- **RECEIVE** - รับสินค้าเข้าคลัง
- **ISSUE** - เบิกสินค้าออกจากคลัง
- **TRANSFER_IN** - โอนเข้า
- **TRANSFER_OUT** - โอนออก
- **ADJUST** - ปรับปรุงสต็อก
- **COUNT** - นับสต็อก

## API Endpoints ที่เกี่ยวข้อง
- `GET/POST /api/warehouses` - จัดการคลังสินค้า
- `GET/POST /api/stock-movements` - บันทึกการเคลื่อนไหวสินค้า
- `GET /api/stock-balances` - ดูยอดคงเหลือสินค้า
- `GET/POST /api/stock/transfers` - โอนสินค้าระหว่างคลัง

## หมายเหตุ
- ระบบจำเป็นต้องมีคลังสินค้าอย่างน้อย 1 คลังก่อนที่จะบันทึกการเคลื่อนไหวสินค้าได้
- สามารถสร้างคลังสินค้าเพิ่มได้ที่เมนู "คลังสินค้า" → "เพิ่มคลังสินค้า"
