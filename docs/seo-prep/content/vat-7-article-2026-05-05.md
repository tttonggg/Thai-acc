## ภาษีมูลค่าเพิ่ม (VAT) คืออะไร?

ภาษีมูลค่าเพิ่ม หรือ Value Added Tax (VAT) คือภาษีที่จัดเก็บจากการบริโภคสินค้าหรือบริการในประเทศ โดยผู้ประกอบการที่จดทะเบียน VAT จะเก็บภาษีจากลูกค้าแล้วนำส่งกรมสรรพากร

**อัตรา VAT มาตรฐานในประเทศไทย ณ ปี 2026 คือ 7%**

ภาษีมูลค่าเพิ่มนี้เกิดขึ้นในทุกขั้นตอนของการผลิตและจำหน่าย แต่ผู้บริโภคปลายทางคือผู้แบกภาระภาษีทั้งหมด ผู้ประกอบการเ� merely act as a collection agent for the government

### ใครต้องจดทะเบียน VAT?

| ประเภท | เกณฑ์ |
|--------|-------|
| VAT ภาคบังคับ | รายได้เกิน 1,800,000 บาท/ปี |
| VAT สมัครใจ | รายได้ต่ำกว่า 1.8 ล้านบาท แต่ต้องการใบเสร็จรับเงินที่มี VAT |

ธุรกิจที่ไม่จดทะเบียน VAT ไม่สามารถออกใบกำกับภาษีให้ลูกค้าได้ ซึ่งอาจสร้างความไม่สะดวกในการทำธุรกิจกับองค์กรใหญ่ที่ต้องการใบเสร็จภาษีอย่างยิ่ง

## VAT 7% คำนวณอย่างไร?

การคำนวณภาษีมูลค่าเพิ่มมี 2 กรณีหลัก:

### 1. ราคาที่รวม VAT แล้ว (Inclusive Price)

หากราคาที่ปรากฏบนสินค้าหรือใบเสร็จรับเงินเป็นราคาที่รวมภาษีแล้ว สามารถแยก VAT ออกได้ดังนี้:

**สูตร:**  
```
VAT = ราคาที่รวม VAT ÷ 1.07 × 0.07
ราคาก่อน VAT = ราคาที่รวม VAT ÷ 1.07
```

**ตัวอย่าง:** ซื้อสินค้าราคา 10,700 บาท (รวม VAT 7%)

| ขั้นตอน | การคำนวณ |
|---------|----------|
| ราคาก่อน VAT | 10,700 ÷ 1.07 = **10,000 บาท** |
| VAT 7% | 10,000 × 0.07 = **700 บาท** |
| ตรวจสอบ | 10,000 + 700 = 10,700 ✅ |

### 2. ราคาที่ไม่รวม VAT (Exclusive Price)

หากราคาเป็นราคาก่อนรวมภาษี การคำนวณจะง่ายกว่า:

**สูตร:**  
```
VAT = ราคาก่อน VAT × 0.07
ราคารวม VAT = ราคาก่อน VAT × 1.07
```

**ตัวอย่าง:** บริการจัดทำบัญชีราคา 50,000 บาท (ไม่รวม VAT)

| ขั้นตอน | การคำนวณ |
|---------|----------|
| VAT 7% | 50,000 × 0.07 = **3,500 บาท** |
| ราคารวม VAT | 50,000 + 3,500 = **53,500 บาท** |

### 3. ตารางเปรียบเทียบ VAT จากยอดต่างๆ

| ราคาก่อน VAT | VAT 7% | ราคารวม VAT |
|-------------|--------|-------------|
| 1,000 | 70 | 1,070 |
| 5,000 | 350 | 5,350 |
| 10,000 | 700 | 10,700 |
| 25,000 | 1,750 | 26,750 |
| 50,000 | 3,500 | 53,500 |
| 100,000 | 7,000 | 107,000 |

## ใครต้องยื่นภาษีมูลค่าเพิ่ม?

ผู้ประกอบการที่จดทะเบียน VAT ทุกรายต้องยื่นแบบแสดงรายการภาษีมูลค่าเพิ่ม **ภายในวันที่ 15 ของเดือนถัดจากเดือนภาษี**

### ประเภทแบบแสดงรายการ

| แบบ | ความหมาย | การยื่น |
|-----|---------|---------|
| **ภ.พ.30** | แบบแสดงรายการภาษีมูลค่าเพิ่ม | ยื่นทุกเดือน |
| **ภ.พ.36** | แบบแสดงรายการภาษีสำหรับผู้ประกอบการส่งออก | ตามที่กำหนด |

### กรณีไม่ยื่นแบบหรือยื่นไม่ทัน

- เบี้ยปรับ 1% ต่อเดือนของจำนวนภาษีที่ต้องยื่น
- อายุความผิด 10 ปี หากพบว่าหลีกเลี่ยงภาษีโดย cork
- ดอกเบี้ยเกินกำหนด 1.25% ต่อเดือน

## ภาษีมูลค่าเพิ่ม 7% สำหรับการนำเข้าและส่งออก

### การนำเข้าสินค้า (Import VAT)

นำเข้าสินค้าจากต่างประเทศต้องเสีย VAT ณ ด่านศุลกากร โดย VAT คำนวณจาก:

```
VAT นำเข้า = (ราคา CIF + อากรขาเข้า + ภาษีนำเข้าอื่นๆ) × 7%
```

**ตัวอย่าง:** นำเข้าสินค้าราคา CIF 500,000 บาท อากรขาเข้า 50,000 บาท

| รายการ | จำนวน |
|--------|-------|
| ราคา CIF | 500,000 |
| อากรขาเข้า (10%) | 50,000 |
| ฐานภาษี | 550,000 |
| **VAT 7%** | **38,500** |

### การส่งออกสินค้า (Export)

สินค้าที่ส่งออกไปต่างประเทศได้รับยกเว้น VAT 0% หากพิสูจน์ได้ว่าสินค้านั้นส่งออกจริง เอกสารที่ต้องใช้:

- ใบสำคัญการส่งออก (Export Declaration)
- บัตรผ่านด่าน (Customs Entry)
- ใบแจ้งราคาสินค้า (Invoice)

## วิธียื่นภาษีมูลค่าเพิ่มออนไลน์ ทีละขั้นตอน

กรมสรรพากรเปิดให้ยื่นภาษีออนไลน์ผ่านระบบ e-Filing ที่ www.rd.go.th โดยมีขั้นตอนดังนี้:

### ขั้นตอนที่ 1: ลงทะเบียนและเข้าสู่ระบบ

1. เข้าเว็บไซต์ www.rd.go.th
2. ไปที่เมนู "e-Filing" > "ลงทะเบียน"
3. กรอกข้อมูลเลขประจำตัวผู้เสียภาษี (13 หลัก)
4. สร้างรหัสผ่านและยืนยันอีเมล
5. เข้าสู่ระบบด้วยเลขประจำตัว + รหัสผ่าน

### ขั้นตอนที่ 2: เตรียมข้อมูลสำหรับยื่น

ก่อนเริ่มกรอกแบบ ต้องเตรียมข้อมูลต่อไปนี้:

- ยอดขายทั้งหมด (รวม VAT และไม่รวม VAT)
- ยอดซื้อทั้งหมด (รวม VAT และไม่รวม VAT)
- VAT ขายที่เก็บได้
- VAT ซื้อที่จ่ายไป
- ภาษีมูลค่าเพิ่มที่ต้องยื่น (ถ้ามี)

### ขั้นตอนที่ 3: กรอกแบบ ภ.พ.30

1. เลือกประเภทแบบแสดงรายการ → "ภ.พ.30"
2. เลือกเดือนภาษีที่ต้องการยื่น
3. กรอกยอดขายและยอดซื้อในช่องที่กำหนด
4. ตรวจสอบยอด VAT ที่ระบบคำนวณอัตโนมัติ
5. กด "บันทึก" และ "ตรวจสอบ" ก่อนส่ง

### ขั้นตอนที่ 4: ชำระภาษีและส่งแบบ

**กรณีมีภาษีต้องชำระ:**
- ชำระผ่านอินเทอร์เน็ตแบงก์กิ้ง (ทุกธนาคาร)
- ชำระผ่านบัตรเครดิต/เดบิต
- ชำระ ณ จุดรับชำระ (เฉพาะกรณีพิเศษ)

**กรณีมีภาษีรอคืน:**
- ระบบจะคำนวณให้โดยอัตโนมัติ
- สามารถขอคืนเงินภาษีได้ภายใน 3 เดือน

> **หมายเหตุ:** หากธุรกิจของคุณมีรายการซื้อขายจำนวนมาก การใช้โปรแกรมบัญชีออนไลน์ที่เชื่อมต่อกับระบบ e-Filing จะช่วยลดความผิดพลาดและประหยัดเวลาได้มาก

## FAQ — คำถามที่พบบ่อยเกี่ยวกับ VAT 7%

### ภาษีมูลค่าเพิ่ม 7% คำนวณอย่างไร?

ภาษีมูลค่าเพิ่ม 7% คำนวณจากราคาสินค้าหรือบริการก่อน VAT สูตรคือ: **VAT = ราคาก่อน VAT × 0.07**  
หากราคารวม VAT แล้ว ใช้สูตร: **VAT = ราคารวม ÷ 1.07 × 0.07**

### ยื่นภาษีมูลค่าเพิ่ม ออนไลน์ ที่ไหน?

ยื่นได้ที่เว็บไซต์กรมสรรพากร www.rd.go.th ในส่วน e-Filing หรือผ่านแอปพลิเคชัน "RD e-Filing" บนสมาร์ทโฟน

### ใครต้องยื่นภาษีมูลค่าเพิ่ม?

ผู้ประกอบการที่จดทะเบียน VAT ทุกรายต้องยื่นแบบ ภ.พ.30 ภายในวันที่ 15 ของเดือนถัดจากเดือนภาษี โดยเกณฑ์การจดทะเบียน VAT คือรายได้เกิน 1.8 ล้านบาทต่อปี

### ภาษีมูลค่าเพิ่ม นำเข้า คิดยังไง?

VAT นำเข้าคำนวณจากราคา CIF + อากรขาเข้า + ภาษีนำเข้าอื่นๆ ทั้งหมด คูณด้วย 7% ยกตัวอย่าง: CIF 500,000 + อากร 50,000 = ฐาน 550,000 × 7% = VAT 38,500 บาท

### VAT 7% และ VAT 10% ต่างกันอย่างไร?

VAT 7% เป็นอัตรามาตรฐานสำหรับสินค้าและบริการทั่วไปในประเทศไทย VAT 10% เคยใช้ในอดีตสำหรับบริการบางประเภท แต่ปัจจุบันอัตรา 10% ยกเลิกไปแล้ว คงเหลือเพียงอัตรา 7% เท่านั้น
