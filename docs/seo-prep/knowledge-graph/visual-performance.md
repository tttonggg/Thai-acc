# Visual Performance Tracking

> Track which visual types drive engagement, CTR, and social shares.

## Visual Types Defined

| Type | Description |
|------|-------------|
| Infographic | Multi-panel visual with data/statistics |
| Comparison Table | Side-by-side product/service comparison |
| Decision Tree | Flowchart for yes/no decision paths |
| Checklist | Numbered action items with checkboxes |
| Screenshot | Annotated app/website UI captures |
| Process Diagram | Step-by-step workflow visualization |
| Chart/Graph | Data visualization (bar, line, pie) |
| Scorecard | Rating matrix with criteria |
