# 📚 คู่มือผู้ใช้ Keerati ERP

## 🎯 แนะนำระบบ

**Keerati ERP** คือ โปรแกรมบัญชีออนไลน์สไตล์พาสเทล ออกแบบมาให้ใช้งานง่าย สวยงาม และครบวงจร รองรับมาตรฐานบัญชีไทย (TFRS)

### ✨ ฟีเจอร์หลัก
- 📊 แดชบอร์ดภาพรวมธุรกิจ
- 📒 บัญชีแยกประเภท (ผังบัญชี)
- 📝 บันทึกรายการบัญชี
- 🧾 ใบกำกับภาษี/ใบเสร็จ
- 💰 จัดการลูกหนี้-เจ้าหนี้
- 📦 สต็อกสินค้าและคลัง
- 🏛️ ภาษีมูลค่าเพิ่ม และภาษีหัก ณ ที่จ่าย
- 👥 เงินเดือนพนักงาน
- 📈 รายงานต่างๆ

---

## 🚀 เริ่มต้นใช้งาน

### 1. เข้าสู่ระบบ
- เปิดเบราว์เซอร์ เข้า URL ของระบบ
- กรอกอีเมลและรหัสผ่าน
- คลิก "เข้าสู่ระบบ"

### 2. บัญชีทดสอบเริ่มต้น
| บทบาท | อีเมล | รหัสผ่าน |
|-------|-------|---------|
| ผู้ดูแลระบบ | admin@thaiaccounting.com | admin123 |
| นักบัญชี | accountant@thaiaccounting.com | acc123 |
| ผู้ใช้ทั่วไป | user@thaiaccounting.com | user123 |
| ผู้ชม | viewer@thaiaccounting.com | viewer123 |

## 🗂️ เมนูหลัก

### 📊 ภาพรวม (Dashboard)
- ดูสถิติยอดขาย ค่าใช้จ่าย กำไร-ขาดทุน
- กราฟแสดงแนวโน้มธุรกิจ
- ลูกหนี้-เจ้าหนี้ค้างชำระ

### 📒 ผังบัญชี
- ดูรายการบัญชีทั้งหมด
- ค้นหาบัญชีตามรหัสหรือชื่อ
- ดูยอดคงเหลือของแต่ละบัญชี

### 📝 บันทึกบัญชี
- สร้างรายการบัญชีใหม่
- บันทึกเดบิต-เครดิต
- แนบเอกสารประกอบ

### 🧾 ใบกำกับภาษี
- สร้างใบกำกับภาษีขาย
- คำนวณภาษีมูลค่าเพิ่ม 7%
- พิมพ์ใบกำกับภาษี PDF

### 👥 ลูกหนี้
- จัดการข้อมูลลูกค้า
- ดูยอดลูกหนี้ค้างชำระ
- บันทึกการรับชำระเงิน

### 🚚 เจ้าหนี้
- จัดการข้อมูลผู้ขาย
- ดูยอดเจ้าหนี้ค้างชำระ
- บันทึกการจ่ายเงิน

### 📦 สต็อกสินค้า
- ดูยอดคงเหลือสินค้า
- บันทึกการรับ-เบิกสินค้า
- โอนย้ายสินค้าระหว่างคลัง

### 🏛️ ภาษี
- **ภาษีมูลค่าเพิ่ม**: รายงานภาษีซื้อ-ขาย
- **ภาษีหัก ณ ที่จ่าย**: พิมพ์ ภ.ง.ด.3, ภ.ง.ด.53

### 👔 เงินเดือน
- จัดการข้อมูลพนักงาน
- คำนวณเงินเดือน ประกันสังคม ภาษี
- พิมพ์สลิปเงินเดือน

## 🎨 การปรับแต่งธีม

1. ดูที่ sidebar ด้านล่าง
2. คลิก **"ปรับแต่งธีม"**
3. เลือกธีมสีที่ชอบ:
   - 🌸 ชมพูพาสเทล (ค่าเริ่มต้น)
   - 🌿 มิ้นท์สดชื่น
   - 💜 ลาเวนเดอร์
   - 🍑 พีชหวาน
   - 🩵 ฟ้าสดใส
   - 🍋 เลมอนสด
   - 🪸 คอรัลพีช
4. เปิด/ปิด **โหมดกลางคืน**
5. ปรับ **ความมนของขอบ** และ **ความเข้มของสี**

## 💡 เคล็ดลับการใช้งาน

### คีย์ลัด (Keyboard Shortcuts)
- `Ctrl + S` = บันทึกฟอร์ม
- `Ctrl + N` = สร้างใหม่
- `Ctrl + F` = ค้นหา
- `ESC` = ปิด Dialog

### การค้นหา
- ใช้ช่องค้นหาที่ด้านบนของแต่ละหน้า
- รองรับการค้นหาด้วยชื่อ รหัส หรือคำสำคัญ

### การ Export
- สามารถ Export ข้อมูลเป็น Excel หรือ PDF ได้
- คลิกปุ่ม "ดาวน์โหลด" ที่มุมขวาของตาราง

### การ Print
- รองรับการพิมพ์ใบกำกับภาษี ใบเสร็จ สลิปเงินเดือน
- ใช้งานได้ทั้ง Print ธรรมดา และ Print เป็น PDF

## ⚙️ การตั้งค่าระบบ (สำหรับ Admin)

### จัดการผู้ใช้
1. ไปที่เมนู **ผู้ดูแลระบบ > จัดการผู้ใช้**
2. คลิก **เพิ่มผู้ใช้**
3. กรอกข้อมูล อีเมล รหัสผ่าน บทบาท
4. คลิก **บันทึก**

### บทบาทผู้ใช้
- **ADMIN**: จัดการทุกอย่างได้
- **ACCOUNTANT**: บันทึกบัญชี ออกรายงาน
- **USER**: ใช้งานทั่วไป
- **VIEWER**: ดูอย่างเดียว

### ตั้งค่าบริษัท
1. ไปที่เมนู **ตั้งค่า**
2. แก้ไขข้อมูลบริษัท
3. อัปโหลดโลโก้
4. ตั้งค่าปีบัญชี

## 🆘 แก้ไขปัญหาเบื้องต้น

### ลืมรหัสผ่าน
ติดต่อผู้ดูแลระบบให้รีเซ็ตรหัสผ่าน

### หน้าเว็บโหลดช้า
- รีเฟรชหน้าเว็บ (F5)
- ล้างแคชเบราว์เซอร์
- ตรวจสอบการเชื่อมต่ออินเทอร์เน็ต

### ข้อมูลไม่บันทึก
- ตรวจสอบว่ากรอกข้อมูลครบถ้วน
- ดูว่ามีข้อความแจ้งเตือนอะไรหรือไม่
- ลองบันทึกใหม่อีกครั้ง

### ไม่สามารถเข้าสู่ระบบได้
- ตรวจสอบอีเมลและรหัสผ่าน
- ตรวจสอบ Caps Lock
- ลองใช้บัญชีอื่น
